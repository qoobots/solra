# 协议定义层（Contracts）技术设计

> contracts/ 的技术选型、工具链与工程规范——基于 Buf 工具链的协议工程化方案

---

## 一、技术选型

| 维度 | 选择 | 理由 |
|------|------|------|
| **IDL语言** | Protocol Buffers 3 (proto3) | 行业标准，多语言生态成熟 |
| **事件Schema** | Apache Avro | 支持Schema演化，适合Kafka生态 |
| **REST规范** | OpenAPI 3.0 | 行业标准REST API描述语言 |
| **代码生成** | Buf (buf.build) | 现代化Proto工具链，替代protoc |
| **版本检查** | buf breaking | 自动检测不兼容变更 |
| **格式化** | buf format | 统一代码风格 |
| **Lint** | buf lint | 命名/结构规范检查 |

---

## 二、工具链配置

### buf.yaml

```yaml
version: v2
modules:
  - path: proto
    name: buf.build/solra/proto
lint:
  use:
    - DEFAULT
  except:
    - PACKAGE_VERSION_SUFFIX
breaking:
  use:
    - FILE
```

### buf.gen.yaml（代码生成）

```yaml
version: v2
plugins:
  - remote: buf.build/protocolbuffers/java
    out: ../services/common/src/generated/java
  - remote: buf.build/community/timostamm-protobuf-ts
    out: ../clients/web/src/generated
  - remote: buf.build/protocolbuffers/python
    out: ../ai-services/shared/generated
  - remote: buf.build/protocolbuffers/cpp
    out: ../core/generated
```

## 四、CI/CD集成

```yaml
# .github/workflows/proto-ci.yml
name: Proto CI
on:
  pull_request:
    paths: ['contracts/proto/**']
jobs:
  lint:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: bufbuild/buf-setup-action@v1
      - run: buf lint
  breaking:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: bufbuild/buf-setup-action@v1
      - run: buf breaking --against 'https://github.com/solra/solra.git#branch=main'
```
