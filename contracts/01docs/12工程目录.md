# 协议定义层（Contracts）工程目录结构

> contracts/ 的完整目录树与各目录职责说明

---

## 一、目录树

```
contracts/
├── 01docs/                         # 本文档目录
│   ├── 01需求分析.md
│   ├── 02产品设计.md
│   ├── 03功能清单.md
│   ├── 04应用设计.md
│   ├── 05技术设计.md
│   ├── 06交互设计.md
│   ├── 07数据设计.md
│   ├── 08安全设计.md
│   ├── 09部署设计.md
│   ├── 10质量设计.md
│   ├── 11API设计.md
│   ├── 12工程目录.md               # 本文
│   └── 13开发进度.md
├── proto/                          # Protocol Buffers 协议定义
│   ├── common/                     # 公共基础类型 (P0)
│   │   ├── common.proto            # UserId/SpaceId/Timestamp
│   │   ├── pagination.proto        # 分页/排序
│   │   └── error.proto             # 统一错误码
│   ├── avt/                        # 虚拟人交互 (P0)
│   ├── spc/                        # 空间消费 (P0)
│   ├── auth/                       # 身份认证 (P0)
│   ├── saf/                        # 内容安全 (P0)
│   ├── crt/                        # 空间创作 (P1)
│   ├── soc/                        # 社交在场 (P1)
│   ├── grw/                        # 用户成长 (P1)
│   ├── not/                        # 通知消息 (P1)
│   └── mon/                        # 商业化 (P2)
├── events/                         # Apache Avro 事件定义
│   ├── space_events.avsc
│   ├── user_events.avsc
│   └── avatar_events.avsc
├── openapi/                        # OpenAPI 3.0 规范
│   ├── services/                   # 服务间 REST API
│   └── client/                     # 客户端 BFF API
├── buf.yaml                        # Buf 顶层配置
├── buf.gen.yaml                    # 多语言代码生成配置
└── buf.lock                        # 依赖版本锁定
```

---

## 二、各目录职责

| 目录 | 职责 | 变更频率 | 审查要求 |
|------|------|---------|---------|
| `proto/common/` | 全项目共享基础类型 | 低 | 架构师审查 |
| `proto/avt/` | 虚拟人交互领域模型 | 高 | 领域负责人审查 |
| `proto/spc/` | 空间消费领域模型 | 高 | 领域负责人审查 |
| `proto/auth/` | 身份认证领域模型 | 中 | 安全审查 |
| `proto/saf/` | 内容安全领域模型 | 中 | 安全审查 |
| `events/` | 领域事件定义 | 中 | 跨域协调 |
| `openapi/` | REST API规范 | 中 | 客户端+服务端双审查 |

---

## 三、命名规范

| 类型 | 规范 | 示例 |
|------|------|------|
| Proto文件 | 小写蛇形 | `avt_service.proto` |
| Avro文件 | 小写蛇形 | `space_events.avsc` |
| OpenAPI文件 | 小写蛇形 | `auth_service.yaml` |
| 目录 | 小写无分隔符 | `avt/` `spc/` |
