# 核心引擎层（Core）工程目录结构

> Solra Core SDK 完整目录树与各模块职责

---

## 一、目录树

```
core/
├── 01docs/                          # 本文档目录
│   ├── 01需求分析.md
│   ├── 02产品设计.md
│   ├── 03功能清单.md
│   ├── 04应用设计.md
│   ├── 05技术设计.md
│   ├── 06交互设计.md
│   ├── 07数据设计.md
│   ├── 08安全设计.md
│   ├── 09部署设计.md
│   ├── 10质量设计.md
│   ├── 11API设计.md
│   ├── 12工程目录.md                # 本文
│   └── 13开发进度.md
├── cmake/                           # CMake 工具链
│   ├── toolchains/
│   │   ├── ios.cmake                # iOS 交叉编译
│   │   ├── android.cmake            # Android NDK 交叉编译
│   │   └── windows.cmake            # Windows MSVC
│   └── FindDependencies.cmake       # 依赖查找
├── include/                         # 公开头文件 (C ABI)
│   └── solra/
│       ├── solra_core.h             # 主入口
│       ├── solra_render.h           # 渲染API
│       ├── solra_inference.h        # 推理API
│       ├── solra_streaming.h        # 流式加载API
│       ├── solra_webrtc.h           # WebRTC API
│       ├── solra_animation.h        # 动画API
│       └── solra_types.h            # 公共类型
├── src/                             # C++17 实现
│   ├── core/                        # 内核 (初始化/生命周期)
│   ├── render/                      # 渲染引擎
│   │   ├── metal/                   # Metal 后端
│   │   ├── vulkan/                  # Vulkan 后端
│   │   └── opengl/                  # OpenGL ES 兜底
│   ├── inference/                   # 端侧推理
│   │   └── llama/                   # llama.cpp 集成
│   ├── streaming/                   # 流式加载
│   ├── webrtc/                      # WebRTC 集成
│   ├── animation/                   # 动画系统
│   └── platform/                    # 平台抽象层
│       ├── ios/
│       ├── android/
│       ├── windows/
│       └── macos/
├── tests/                           # 测试
│   ├── unit/                        # 单元测试 (GTest)
│   ├── integration/                 # 集成测试
│   └── benchmark/                   # 性能基准 (GBenchmark)
├── third_party/                     # 第三方库 (git submodule)
│   └── llama.cpp/                   # 端侧推理引擎
├── CMakeLists.txt                   # 根构建配置
└── vcpkg.json                       # 依赖清单
```

---

## 二、模块职责

| 模块 | 目录 | 职责 | 优先级 |
|------|------|------|--------|
| 内核 | `src/core/` | SDK初始化/销毁/生命周期 | P0 |
| 渲染引擎 | `src/render/` | 3D场景渲染(Metal/Vulkan/GLES) | P0 |
| 推理引擎 | `src/inference/` | 端侧AI推理(llama.cpp) | P0 |
| 流式加载 | `src/streaming/` | 资产分块下载+缓存 | P0 |
| WebRTC | `src/webrtc/` | P2P音视频数据通道 | P1 |
| 动画 | `src/animation/` | 表情/口型/IK动画 | P1 |
| 平台抽象 | `src/platform/` | iOS/Android/Windows/macOS平台层 | P0 |
