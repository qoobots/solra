# 核心引擎层（Core）部署设计

> Core SDK 的"部署"即各平台二进制产物的编译、打包与分发

---

## 一、平台产物

| 平台 | 产物 | 分发方式 |
|------|------|---------|
| iOS | `SolraCore.framework` (arm64) | 嵌入 App Bundle |
| Android | `libsolracore.so` (arm64-v8a) | jniLibs 目录 |
| Windows | `SolraCore.lib` + DLL | 开发机本地编译 |
| macOS | `libSolraCore.a` (arm64/x86_64) | CI 编译 |

---

## 二、编译环境

| 平台 | 编译器 | 最低版本 |
|------|--------|---------|
| iOS | Apple Clang 15+ | Xcode 15+ |
| Android | NDK r26+ (Clang 17) | API 26+ |
| Windows | MSVC 2022 | Windows 10+ |
| macOS | Apple Clang 15+ | macOS 13+ |
