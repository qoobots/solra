# 核心引擎层（Core）数据设计

> Core SDK 内部数据结构与内存管理策略

---

## 一、核心数据结构

### 1.1 渲染相关

```cpp
// 场景图节点
struct SceneNode {
    NodeId id;
    Transform local_transform;
    std::vector<SceneNode*> children;
    MeshHandle mesh;
    MaterialHandle material;
    bool visible;
};

// GPU资源句柄（不透明）
using MeshHandle = uint32_t;
using TextureHandle = uint32_t;
using MaterialHandle = uint32_t;
using ShaderHandle = uint32_t;
```

### 1.2 推理相关

```cpp
struct InferenceRequest {
    std::string conversation_id;
    std::string user_input;
    InferenceConfig config; // temperature, top_p, max_tokens
};

struct InferenceToken {
    std::string token;
    float probability;
    bool is_final;
};
```

### 1.3 流式加载

```cpp
struct AssetChunk {
    std::string asset_id;
    uint32_t chunk_index;
    uint32_t total_chunks;
    LODLevel lod; // LOD0(全精度) / LOD1(中精度) / LOD2(低精度)
    std::vector<uint8_t> data;
};

enum class LODLevel { HIGH, MEDIUM, LOW };
```

## 二、内存管理

| 策略 | 说明 |
|------|------|
| **GPU内存** | 独立池管理，Metal/Vulkan各自实现 |
| **模型内存** | mmap加载，多进程共享 |
| **资产缓存** | LRU淘汰，上限256MB |
| **推理KV Cache** | 固定预分配，避免动态扩容 |

---

## 三、持久化

| 数据类型 | 存储位置 | 格式 |
|---------|---------|------|
| 资产缓存 | 沙盒 `Library/Caches/solra/` | 二进制blob |
| 模型文件 | 沙盒 `Library/Models/` | GGUF量化格式 |
| 对话历史 | 沙盒 `Library/Application Support/` | Protocol Buffers |
| 用户偏好 | NSUserDefaults / SharedPreferences | Key-Value |
