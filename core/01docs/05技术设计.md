# 核心引擎层（Core）技术设计

> Solra Core SDK 技术选型与工程规范——C++17 + CMake + vcpkg 跨平台构建方案

---

## 一、技术选型

| 维度 | 选择 | 理由 |
|------|------|------|
| **语言** | C++17 | 跨平台性能+现代C++特性 |
| **构建系统** | CMake 3.24+ | 行业标准跨平台构建 |
| **包管理** | vcpkg | 微软维护，iOS/Android/Windows支持 |
| **渲染后端** | Metal(iOS) / Vulkan(Android) / GLES3(兜底) | 平台最优性能 |
| **推理引擎** | llama.cpp | 轻量级C++端侧推理 |
| **WebRTC** | libwebrtc (裁剪版) | Google官方C++实现 |
| **3D数学** | GLM | 头文件库，语法兼容GLSL |
| **测试** | Google Test + Google Benchmark | C++测试标配 |
| **格式化** | clang-format | 统一代码风格 |
| **静态分析** | clang-tidy | 编译时检查 |

---

## 二、构建系统设计

### CMake 结构

```
core/
├── CMakeLists.txt            # 根配置
├── cmake/
│   ├── toolchains/
│   │   ├── ios.cmake         # iOS交叉编译
│   │   └── android.cmake     # Android NDK交叉编译
│   └── FindDependencies.cmake
├── src/
│   ├── render/CMakeLists.txt
│   ├── inference/CMakeLists.txt
│   ├── streaming/CMakeLists.txt
│   ├── webrtc/CMakeLists.txt
│   └── animation/CMakeLists.txt
└── vcpkg.json               # 依赖清单
```

## 三、编码规范

| 规则 | 说明 |
|------|------|
| 命名空间 | `solra::render`, `solra::inference` |
| 头文件组织 | `include/solra/{module}/{class}.h` |
| RAII | 所有资源用RAII管理 |
| 无异常跨FFI | 对外C接口不使用异常 |
| 智能指针 | 内部用`std::unique_ptr/shared_ptr` |
