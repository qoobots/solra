## 一、gRPC 服务间调用

```java
// avt-service 调用 auth-service 验证Token
@GrpcClient("auth-service")
private AuthServiceGrpc.AuthServiceBlockingStub authStub;

public void verifyToken(String token) {
    VerifyTokenResponse response = authStub.verifyToken(
        VerifyTokenRequest.newBuilder()
            .setToken(token)
            .build()
    );
}
```

## 二、事件发布/订阅

```java
// avt-service 发布事件
@Service
public class AvatarEventPublisher {
    private final KafkaTemplate<String, Object> kafka;
    
    public void publishAvatarSpoke(AvatarSpokeEvent event) {
        kafka.send("avatar.spoke", event.conversationId(), event);
    }
}

// soc-service 消费事件
@Component
public class AvatarSpokeConsumer {
    @KafkaListener(topics = "avatar.spoke")
    public void onAvatarSpoke(AvatarSpokeEvent event) {
        // 更新用户在场状态
    }
}
```

## 三、REST API 规范

### 3.1 统一响应格式

```json
{
    "code": 0,
    "message": "success",
    "data": { ... },
    "trace_id": "uuid"
}
```

### 3.2 分页格式

```json
{
    "code": 0,
    "data": {
        "items": [...],
        "total": 100,
        "has_more": true,
        "next_cursor": "eyJ..."
    }
}
```
