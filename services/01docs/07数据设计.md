# 微服务层（Services）数据设计

> services/ 数据库设计——PostgreSQL 主存储 + Redis 缓存 + Elasticsearch 搜索

---

## 一、数据库分配

| 服务 | 数据库名 | 存储类型 |
|------|---------|---------|
| auth-service | `solra_auth` | PostgreSQL |
| saf-service | `solra_saf` | PostgreSQL |
| avt-service | `solra_avt` | PostgreSQL + MongoDB (对话) |
| spc-service | `solra_spc` | PostgreSQL + Elasticsearch (搜索) |
| soc-service | `solra_soc` | PostgreSQL + Redis (在场) |
| grw-service | `solra_grw` | PostgreSQL |
| not-service | `solra_not` | PostgreSQL |
| crt-service | `solra_crt` | PostgreSQL + MongoDB (资产) |
| mon-service | `solra_mon` | PostgreSQL |

---

## 二、表设计规范

### 2.1 公共字段

```sql
-- 每张表都包含
id          UUID PRIMARY KEY DEFAULT gen_random_uuid(),
created_at  TIMESTAMP NOT NULL DEFAULT NOW(),
updated_at  TIMESTAMP NOT NULL DEFAULT NOW(),
version     INTEGER NOT NULL DEFAULT 0  -- 乐观锁
```

### 2.2 索引策略

| 索引类型 | 用途 |
|---------|------|
| B-tree (默认) | 等值/范围查询 |
| GIN | JSONB/全文搜索 |
| GiST | 地理位置查询 |
| Partial Index | 条件索引 (WHERE deleted_at IS NULL) |

---

## 三、缓存策略

| 缓存对象 | TTL | 策略 |
|---------|-----|------|
| 用户Session | 30分钟 | Cache-Aside |
| 空间元数据 | 5分钟 | Cache-Aside + 主动失效 |
| 推荐结果 | 1分钟 | Write-Behind |
| 安全评分 | 30秒 | Read-Through |

---

## 四、数据保护

| 字段类型 | 保护方式 |
|---------|---------|
| 密码 | bcrypt 哈希 |
| 手机号 | AES-256 加密存储 |
| 邮箱 | AES-256 加密存储 |
| 身份证 | 不存储，仅存哈希 |
| Token | 不存储原始Token，仅存SHA-256 |
