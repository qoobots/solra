# 客户端层（Clients）数据设计

> 客户端本地数据存储与缓存策略

---

## 一、本地存储

| 数据类型 | iOS | Android | Web |
|---------|-----|---------|-----|
| 用户偏好 | UserDefaults | DataStore(Preferences) | localStorage |
| 对话历史 | CoreData(SQLite) | Room(SQLite) | IndexedDB |
| 缓存资产 | FileManager(Caches) | cacheDir | CacheStorage |
| 敏感Token | KeyChain | EncryptedSharedPrefs | httpOnly Cookie |
| 模型文件 | AppGroup(共享) | internalStorage | N/A |

---

## 二、缓存策略

| 缓存对象 | TTL | 策略 |
|---------|-----|------|
| 空间预览卡片 | 5分钟 | 内存缓存 |
| 3D资产纹理 | 按LRU | 磁盘缓存 256MB上限 |
| 推荐列表 | 3分钟 | 内存缓存+分页 |
| 用户Profile | 10分钟 | 单例内存缓存 |

---

## 三、离线支持

| 场景 | 能力 |
|------|------|
| 3D渲染 | 缓存空间可离线进入 |
| 对话 | 离线降级为本地SLM推理 |
| 推荐 | 离线展示缓存内容 |
| 社交 | 需联网 |
