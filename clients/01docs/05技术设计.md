# 客户端层（Clients）技术设计

> clients/ 技术选型与工程规范——三端统一架构原则

---

## 一、技术选型

### iOS

| 维度 | 选择 |
|------|------|
| 语言 | Swift 5.9 |
| UI | SwiftUI |
| 导航 | NavigationStack |
| 依赖管理 | SPM |
| 构建 | Xcode 15 |
| Core SDK | SolraCore.framework |

### Android

| 维度 | 选择 |
|------|------|
| 语言 | Kotlin 2.0 |
| UI | Jetpack Compose |
| 导航 | Compose Navigation |
| 依赖管理 | Gradle Version Catalog |
| 构建 | Gradle 8.x KTS |
| Core SDK | libsolracore.so (JNI) |

### Web

| 维度 | 选择 |
|------|------|
| 语言 | TypeScript 5 |
| UI | React 18 |
| 状态管理 | Zustand |
| 3D渲染 | Three.js / Babylon.js |
| 构建 | Vite |
| 包管理 | pnpm |

---

## 二、编码规范

| 规则 | iOS | Android | Web |
|------|-----|---------|-----|
| 格式化 | SwiftFormat | ktlint | Prettier |
| Lint | SwiftLint | detekt | ESLint |
| 测试 | XCTest | JUnit5 | Vitest |
| UI测试 | XCUITest | Compose Test | Playwright |

---

## 三、Core SDK 桥接

```swift
// iOS Bridge
import SolraCore

let handle = SolraCoreInit(config)
SolraRenderCreateScene(handle, sceneId)
```

```kotlin
// Android JNI Bridge
external fun solraCoreInit(config: SolraConfig): Long
external fun solraRenderCreateScene(handle: Long, sceneId: String)
```
