# 客户端层（Clients）需求分析

> 索拉四端客户端（iOS/Android/Web/Windows Desktop）——承载3D空间交互、AI虚拟人对话、流式内容消费

---

## 一、需求背景

clients/ 是用户直接接触的终端产品，承载索拉"进入空间"范式跃迁的完整体验。

### 1.1 客户端全景

| 客户端 | 平台 | 优先级 | 语言/框架 |
|--------|------|--------|----------|
| iOS | iPhone/iPad | P1 | Swift 5.9 + SwiftUI |
| Android | 主流厂商 | P1 | Kotlin 2.0 + Compose |
| Web | 现代浏览器 | P2 | TypeScript 5 + Vue 3 |
| Windows Desktop | Windows 10/11 | P2 | Rust + Tauri 2 + Vue 3 |

> **架构决策（2026-06-30）**：Web 客户端定位调整为轻量访问端（不承担复杂 3D 渲染）。大型 3D 社交游戏对原生 GPU 利用率、64-bit 内存、端侧 AI 推理的需求，由 Windows 桌面客户端满足。桌面端通过 Tauri 2 壳层 + Rust FFI 直接调用 C++17 Core SDK，UI 层 100% 复用 Web 端 Vue 3 代码。

## 二、功能需求

### 2.1 iOS 客户端 [P1]

| 编号 | 功能 | 说明 |
|------|------|------|
| CLI-IOS-01 | 空间推荐流 | 2D卡片→3D预览→全沉浸空间 |
| CLI-IOS-02 | 3D空间渲染 | Metal + Core SDK集成 |
| CLI-IOS-03 | AI虚拟人对话 | 文本/语音对话 |
| CLI-IOS-04 | 用户认证 | 注册/登录/OAuth |
| CLI-IOS-05 | 个人空间 | 我的空间管理 |
| CLI-IOS-06 | 设置与隐私 | 偏好/安全/隐私 |

### 2.2 Android 客户端 [P1]

| 编号 | 功能 | 说明 |
|------|------|------|
| CLI-AND-01 | 空间推荐流 | 同iOS |
| CLI-AND-02 | 3D空间渲染 | Vulkan + Core SDK JNI |
| CLI-AND-03 | AI虚拟人对话 | 文本/语音对话 |
| CLI-AND-04 | 用户认证 | 注册/登录/OAuth |
| CLI-AND-05 | 个人空间 | 我的空间管理 |

### 2.3 Web 客户端 [P2] — 轻量访问端

| 编号 | 功能 | 说明 |
|------|------|------|
| CLI-WEB-01 | 空间发现浏览 | Feed流、空间详情、搜索 |
| CLI-WEB-02 | 用户管理 | 登录、个人主页、消息中心 |
| CLI-WEB-03 | 轻量空间创作 | Web编辑器（不含3D预览） |
| CLI-WEB-04 | 商城/排行榜 | 虚拟物品浏览、榜单 |

### 2.4 Windows 桌面客户端 [P2] 🆕 — 主力 3D 体验平台

| 编号 | 功能 | 说明 |
|------|------|------|
| CLI-DSK-01 | 3D 空间渲染 | OpenGL/Vulkan + Core SDK 原生渲染 |
| CLI-DSK-02 | AI虚拟人对话 | 端侧 SLM (llama.cpp) + 云端 LLM |
| CLI-DSK-03 | WebRTC 音视频 | P2P 数据通道 + 空间音频 |
| CLI-DSK-04 | 流式加载 | 资产分块下载 + 渐进渲染 |
| CLI-DSK-05 | 空间推荐流 | Feed 流 + 预览卡片 |
| CLI-DSK-06 | 用户认证 | 注册/登录/OAuth |
| CLI-DSK-07 | 个人空间管理 | 我的空间 + 虚拟人配置 |
| CLI-DSK-08 | 端侧推理 | ONNX Runtime 本地推理 |
| CLI-DSK-09 | 离线缓存 | 已缓存空间无网络可进入 |

## 三、非功能需求

| 维度 | iOS | Android | Web |
|------|-----|---------|-----|
| 最低版本 | iOS 15+ | Android 8+ | Chrome 90+ |
| 包体积 | <200MB | <150MB | — |
| 帧率 | ≥30fps | ≥30fps | ≥30fps |
| 冷启动 | <3s | <3s | <3s |
| 离线支持 | 有限 (缓存) | 有限 (缓存) | 需联网 |
