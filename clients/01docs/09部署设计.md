# 客户端层（Clients）部署设计

> 客户端打包、分发、热更新策略

---

## 一、iOS 打包分发

```
Xcode Archive → IPA
  → App Store Connect
    → TestFlight (内测)
    → App Store (正式发布)
  → Fastlane 自动化
```

### 构建配置

| 方案 | Debug | Release |
|------|-------|---------|
| 优化 | -O0 | -Os |
| 符号 | 完整 | dSYM上传 |
| 日志 | 详细 | 仅错误 |

---

## 二、Android 打包分发

```
Gradle assemble → AAB/APK
  → Google Play Console
    → Internal Test → Closed → Open → Production
  → Fastlane 自动化
```

---

## 三、Web 部署

```
Vite build → 静态资源
  → CDN (CloudFront / EdgeOne)
  → 全球边缘分发
```

---

## 四、热更新策略

| 类型 | 方案 | 平台 |
|------|------|------|
| 3D资产 | CDN增量下发 | iOS/Android |
| AI模型 | OTA分发热更新 | iOS/Android |
| Web资源 | 缓存策略+版本号 | Web |
| Native代码 | App Store/Play Store | iOS/Android |
