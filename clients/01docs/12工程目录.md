# 客户端层（Clients）工程目录结构

> clients/ 完整目录树与各客户端职责

---

## 一、目录树

```
clients/
├── 01docs/                          # 本文档目录
│   ├── 01需求分析.md
│   ├── 02产品设计.md
│   ├── 03功能清单.md
│   ├── 04应用设计.md
│   ├── 05技术设计.md
│   ├── 06交互设计.md
│   ├── 07数据设计.md
│   ├── 08安全设计.md
│   ├── 09部署设计.md
│   ├── 10质量设计.md
│   ├── 11API设计.md
│   ├── 12工程目录.md                # 本文
│   └── 13开发进度.md
├── ios/                             # iOS客户端 [P1]
│   ├── Solra.xcodeproj
│   ├── Solra/
│   │   ├── App/                     # App入口
│   │   ├── Views/                   # SwiftUI视图
│   │   │   ├── Discover/            # 发现页
│   │   │   ├── Space/               # 空间页
│   │   │   ├── Profile/             # 个人页
│   │   │   └── Common/              # 公共组件
│   │   ├── ViewModels/              # 状态管理
│   │   ├── Services/                # 业务服务
│   │   ├── CoreSDK/                 # C++ SDK桥接
│   │   ├── Networking/              # 网络层
│   │   └── Resources/               # 资源
│   └── fastlane/                    # CI/CD
├── android/                         # Android客户端 [P1]
│   ├── app/
│   │   └── src/main/kotlin/solra/
│   │       ├── ui/                  # Compose UI
│   │       ├── viewmodel/           # ViewModels
│   │       ├── domain/              # UseCases
│   │       ├── data/                # Repository
│   │       └── core/                # JNI桥接
│   └── fastlane/
└── web/                             # Web客户端 [P2]
    ├── src/
    │   ├── components/              # React组件
    │   ├── pages/                   # 页面
    │   ├── stores/                  # Zustand状态
    │   ├── api/                     # API客户端
    │   └── renderer/                # WebGL渲染
    └── vite.config.ts
```

---

## 二、客户端职责

| 客户端 | 优先级 | 职责 |
|--------|--------|------|
| ios/ | P1 | SwiftUI+Metal渲染完整体验 |
| android/ | P1 | Compose+Vulkan渲染完整体验 |
| web/ | P2 | React+WebGL轻量入口+编辑器 |
