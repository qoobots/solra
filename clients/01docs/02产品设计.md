# 客户端层（Clients）产品设计

> 索拉客户端产品形态——空间优先的3D社交终端体验

---

## 一、产品形态

| 客户端 | 产品定位 | 核心差异 |
|--------|---------|---------|
| iOS | 第一优先级体验平台 | Metal原生渲染+CoreML推理 |
| Android | 最大用户量平台 | Vulkan渲染+NNAPI推理 |
| Web | 轻量入口+编辑器 | WebGL+无需安装 |

---

## 二、统一技术栈

| 维度 | iOS | Android | Web |
|------|-----|---------|-----|
| 语言 | Swift 5.9 | Kotlin 2.0 | TypeScript 5 |
| UI | SwiftUI | Compose | React 18 |
| 渲染 | Metal 3 | Vulkan | WebGL 2.0 |
| 网络 | URLSession | OkHttp | fetch/SSE |
| 存储 | CoreData | Room | IndexedDB |
| Core SDK | SolraCore.framework | libsolracore.so | N/A (Web独立) |
| 构建 | Xcode | Gradle KTS | Vite |

---

## 三、导航架构

```
App 入口
├── Tab1: 发现 (推荐流)
│   ├── 空间预览卡片
│   └── → 全屏3D空间
├── Tab2: 我的空间
│   ├── 个人空间管理
│   └── 虚拟人配置
├── Tab3: 消息/通知
└── Tab4: 我的 (设置/隐私)
```

---

## 四、Core SDK 集成

```
iOS:
  SwiftUI → Swift Bridge → SolraCore.framework → C++17

Android:
  Compose → Kotlin Bridge (JNI) → libsolracore.so → C++17

Web:
  React → WebGL/WebRTC (浏览器原生)
```
