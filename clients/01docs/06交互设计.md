# 客户端层（Clients）交互设计

> 索拉客户端交互设计——空间优先、虚拟人是界面

---

## 一、核心交互原则

| 原则 | 移动端实现 | Web端实现 |
|------|----------|----------|
| 空间优先 | 全屏3D视图为主画布 | WebGL全屏渲染 |
| 虚拟人是界面 | 对话气泡浮于空间之上 | 对话面板侧边栏 |
| 物理隐喻 | 手势驱动3D交互 | 鼠标/触摸 |
| 渐进沉浸 | 卡片→预览→全屏 | 卡片→预览→全屏 |
| 退出即推进 | 退出动画+下一个空间预览 | 退出动画+推荐 |

---

## 二、手势系统

### 2.1 空间浏览手势

```
单指滑动 → 旋转视角
双指缩放 → 拉近/拉远
双击     → 聚焦到虚拟人
上划     → 退出空间
长按     → 呼出选项菜单
```

### 2.2 对话交互

```
点击虚拟人 → 开始对话
输入文字    → 发送消息
语音输入    → 按住录音
```

---

## 三、页面流转

```
发现页(推荐流)
  → 点击卡片 → 空间预览(轻3D)
    → 点击进入 → 全屏3D空间
      → 对话 → 表情+口型+语音
      → 上划 → 退出过渡(动画)
        → 下一个空间预览卡片
```
