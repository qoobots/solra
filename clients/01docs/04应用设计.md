# 客户端层（Clients）应用架构设计

> clients/ 在索拉架构中的位置——用户终端层，消费所有后端能力

---

## 一、架构定位

```
clients/ios (SwiftUI) ─┐
clients/android (Compose) ─┤
                            ├→ Core SDK (C++17) → GPU/NPU
                            │
                            ├→ APISIX Gateway → services/ + ai-services/
                            │
                            └→ CDN → 3D资产/模型文件
```

---

## 二、iOS 架构

```
┌─────────────────────────┐
│ SwiftUI Views           │ ← UI层
├─────────────────────────┤
│ ViewModels (Observable) │ ← 状态管理
├─────────────────────────┤
│ Services Layer          │ ← 业务逻辑
│ ├── AuthService         │
│ ├── SpaceService        │
│ └── AvatarService       │
├─────────────────────────┤
│ SolraCoreSDK Bridge     │ ← C++ FFI
├─────────────────────────┤
│ Networking (gRPC/REST)  │
└─────────────────────────┘
```

---

## 三、Android 架构

```
┌─────────────────────────┐
│ Compose UI              │
├─────────────────────────┤
│ ViewModels              │
├─────────────────────────┤
│ UseCases / Repositories │ ← Clean Architecture
├─────────────────────────┤
│ SolraCoreSDK JNI        │
├─────────────────────────┤
│ Networking              │
└─────────────────────────┘
```

---

## 四、Web 架构

```

├─────────────────────────┤
```
