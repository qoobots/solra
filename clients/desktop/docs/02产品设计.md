## 1. 产品形态

### 1.1 产品定位

Solra Desktop 是面向桌面平台的**全功能 3D 元宇宙客户端**，定位为索拉生态的 **3D 体验主入口** 和 **创作者工作台**。

### 1.2 核心差异点（vs 移动端）

| 维度 | Desktop | Mobile (iOS/Android) |
|------|---------|---------------------|
| 3D 渲染 | 原生 GPU（Vulkan/OpenGL/Metal） | 降级 3D / 2.5D |
| 输入方式 | 键鼠 + 手柄 | 触控 + 手势 |
| 屏幕空间 | 1280x800+ 大屏 | 375-430pt 小屏 |
| 多任务 | 多窗口、多视口 | 单任务 |
| 创作能力 | 完整编辑工具 | 消费为主 |
| 推理能力 | 本地 GPU 加速推理 | 云端推理 |
| 存储空间 | 几乎无限 | 有限 |

## 2. 技术栈

| 层级 | 技术 | 版本 | 选型理由 |
|------|------|------|---------|
| **壳层** | Tauri 2 | 2.x | 轻量（<5MB）、Rust 后端、原生窗口管理 |
| **UI 层** | Vue 3 + TypeScript | 3.4+ | 响应式 UI、与 Web 端共享组件 |
| **UI 组件库** | Element Plus | 2.5+ | 成熟的企业级 Vue 3 组件库 |
| **3D 降级渲染** | Three.js | 0.185 | WebGL 2.0 降级渲染路径 |
| **3D 主力渲染** | Core SDK (C++17) | - | Vulkan/OpenGL/Metal 原生渲染 |
| **Rust 桥接** | libloading + FFI | - | 动态加载 Core SDK DLL |
| **构建工具** | Vite | 5.x | 快速 HMR、ESBuild 打包 |
| **测试** | Vitest + happy-dom | 2.x | 兼容 Vite 的单元测试 |
| **网络** | reqwest + axios | - | Rust 后端 + JS 前端双通道 |
| **状态管理** | Pinia | 2.x | Vue 3 官方状态管理 |
| **CSS** | SCSS | - | 嵌套、变量、Mixin |

## 4. 关键页面设计

### 4.1 主视图（3D 空间）

- **左侧面板**：可折叠侧边栏（280px），空间信息、在线用户、快捷操作
- **中央区域**：3D 视口，占据主要窗口面积
- **底部状态栏**：FPS、GPU 后端、网络延迟、在线人数
- **浮动元素**：对话气泡、通知 Toast、虚拟人对话面板

### 4.2 空间浏览

- **布局**：网格 / 列表双视图
- **筛选**：热门、最新、好友推荐、分类标签
- **搜索**：空间名称、描述、标签模糊搜索
- **预览**：空间缩略图 + 简介

### 4.3 设置面板

- **图形设置**：渲染后端选择（Vulkan/OpenGL/Metal/自动）、分辨率缩放、帧率限制、VSync、抗锯齿、阴影质量
- **音频设置**：输入/输出设备、空间音频开关
- **账号设置**：个人信息、隐私、通知偏好
- **快捷键**：可自定义快捷键绑定
- **关于**：版本号、更新检查、开源许可

### 4.4 对话面板

- **AI 对话**：与空间内 NPC/虚拟人文字对话
- **流式显示**：LLM Token 逐字输出
- **语音输入**：语音转文字（未来）

## 5. Core SDK 集成路径

```
用户操作
  ↓
Vue 3 组件 (RenderViewport.vue)
  ↓  Tauri IPC invoke()
Rust IPC 层 (render_cmd.rs)
  ↓  函数调用
Rust Core 桥接层 (core/render.rs)
  ↓  FFI (libloading)
Core SDK C ABI (libsolracore.dll/.so/.dylib)
  ↓
C++17 渲染引擎 (Vulkan/OpenGL/Metal)
```

**降级路径**：
```
Core SDK 不可用
  ↓
Rust 返回 error → IPC 返回 null
  ↓
前端检测 → 切换 Three.js WebGL Renderer
  ↓
Canvas 内 3D 渲染（有限效果）
```

## 6. 与 Web 客户端的关系

Desktop 客户端与 Web 客户端 (`clients/web`) **UI 层 100% 复用**：

- 共享 Vue 3 组件库（`@solra/ui-components`）
- 共享状态管理逻辑（Pinia stores）
- 共享路由结构
- 差异仅在于：
  - Desktop 有 Core SDK 渲染通道
  - Desktop 有 Rust IPC 命令（vs Web 的 HTTP API）
  - Desktop 有原生窗口能力（系统托盘、文件关联等）
