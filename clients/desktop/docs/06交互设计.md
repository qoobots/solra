# 06 交互设计 — Solra 桌面客户端

> **文档状态**：初稿  
> **最后更新**：2026-06-30  
> **适用版本**：Solra Desktop v0.1.0

---

## 1. 交互原则

### 1.1 核心原则

| 原则 | 说明 |
|------|------|
| **空间优先** | 3D 空间是主界面，UI 面板为辅助叠加层 |
| **键鼠为主** | 桌面端默认交互方式，支持自定义快捷键 |
| **即时反馈** | 所有操作 100ms 内给出视觉/听觉反馈 |
| **渐进沉浸** | 默认显示 UI，可按 Tab 进入沉浸模式（全屏 3D） |
| **非模态优先** | 尽量避免模态对话框，使用浮动面板 |

### 1.2 输入设备支持

| 设备 | 用途 |
|------|------|
| 鼠标 | 3D 视角控制（旋转/缩放/平移）、UI 点击 |
| 键盘 | 快捷键、WASD 移动、文字输入 |
| 游戏手柄 | 3D 空间漫游（未来） |
| 触控板 | 双指缩放、三指旋转（macOS 优化） |

## 2. 3D 空间交互

### 2.1 相机控制

| 操作 | 鼠标 | 键盘 |
|------|------|------|
| 旋转视角 | 左键拖拽 | ← → ↑ ↓ |
| 缩放 | 滚轮 | +/- |
| 平移 | 中键拖拽 / Shift+左键 | WASD |
| 聚焦物体 | 双击物体 | F |
| 重置视角 | - | Home |
| 第一人称 | - | Tab（切换） |

### 2.2 对象交互

| 操作 | 行为 |
|------|------|
| 悬停 | 高亮轮廓（Outline Effect） |
| 单击 | 选中对象，显示信息面板 |
| 双击 | 交互（开门/拾取/对话） |
| 右键 | 上下文菜单（详情/传送/分享） |

### 2.3 沉浸模式

- **进入**：按 Tab 键，隐藏所有 UI 面板
- **退出**：按 Tab 或 Esc
- **半沉浸**：仅显示底部状态栏
- **HUD 信息**：空间名称、在线人数、FPS（半透明）

## 3. UI 交互

### 3.1 侧边栏

| 行为 | 说明 |
|------|------|
| 展开/折叠 | 点击汉堡菜单图标，或按 `Ctrl+B` |
| 宽度调整 | 拖拽侧边栏右边缘（240px - 400px） |
| 导航切换 | 点击导航项，右侧内容区切换 |
| 活动指示 | 当前路由高亮 + 左侧蓝色指示条 |

### 3.2 设置面板

| 设置项 | 交互方式 |
|------|---------|
| 渲染后端 | 下拉选择（自动/Vulkan/OpenGL/Metal） |
| 分辨率缩放 | 滑块（50% - 200%，步长 10%） |
| 帧率限制 | 下拉选择（30/60/120/无限制） |
| VSync | 开关 |
| 抗锯齿 | 下拉选择（关/MSAA 2x/4x/TAA） |
| 阴影质量 | 下拉选择（低/中/高） |
| 音频设备 | 下拉选择（自动检测设备列表） |
| 快捷键 | 可编辑表格，支持按键捕获 |

### 3.3 空间浏览

| 交互 | 说明 |
|------|------|
| 视图切换 | 网格视图 ⇄ 列表视图 |
| 搜索 | 实时搜索，输入即过滤 |
| 筛选 | 标签多选（热门/最新/好友推荐） |
| 排序 | 热门/最新/人数 |
| 进入空间 | 单击卡片 → 加载 3D 空间 |

### 3.4 对话面板（AI Chat）

| 交互 | 说明 |
|------|------|
| 输入框 | 底部固定，支持多行 |
| 发送 | Enter 发送，Shift+Enter 换行 |
| 流式显示 | LLM 回复逐字显示 |
| 停止生成 | 点击停止按钮中断推理 |
| 历史记录 | 滚动加载历史消息 |

## 4. 快捷键

### 4.1 全局快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+B` | 切换侧边栏 |
| `Ctrl+,` | 打开设置 |
| `Ctrl+H` | 打开调试面板 |
| `Ctrl+Q` | 退出应用 |
| `F11` | 全屏切换 |
| `Esc` | 关闭当前面板/对话框 |

### 4.2 3D 视口快捷键

| 快捷键 | 功能 |
|--------|------|
| `W/A/S/D` | 平移视角 |
| `Q/E` | 上升/下降 |
| `F` | 聚焦选中物体 |
| `Home` | 重置视角 |
| `Tab` | 沉浸模式切换 |
| `1/2/3/4` | 预设视角 |

### 4.3 自定义快捷键

设置面板中支持以下操作：
- 查看当前所有快捷键绑定
- 修改绑定（点击后捕获按键）
- 重置为默认
- 导入/导出配置文件

## 5. 反馈机制

### 5.1 加载状态

| 场景 | 反馈 |
|------|------|
| 空间加载中 | 3D 视口显示加载动画 + 进度条 |
| 资源下载 | 状态栏显示下载进度 |
| LLM 推理中 | 对话面板显示思考动画 |
| 操作处理中 | 按钮显示 loading 状态 |

### 5.2 错误提示

| 类型 | 显示方式 |
|------|---------|
| 一般错误 | 顶部 Toast 通知（3s 自动消失） |
| 严重错误 | 模态对话框，需确认 |
| 渲染降级 | 状态栏持续显示降级标记 |
| 网络断开 | 状态栏红色断开图标 |

### 5.3 状态指示

| 状态 | 指示方式 |
|------|---------|
| Core SDK 状态 | 状态栏显示 GPU 后端名称 |
| 网络状态 | 状态栏绿/黄/红圆点 |
| 帧率 | 状态栏实时 FPS 数字 |
| 在线人数 | 状态栏人数图标 |
| 麦克风状态 | 状态栏麦克风图标（开/关） |

## 6. 窗口管理

### 6.1 窗口状态

| 属性 | 默认值 | 可配置 |
|------|--------|--------|
| 初始大小 | 1280 x 800 | ✅ |
| 最小大小 | 1024 x 640 | ❌ |
| 最大大小 | 无限制 | ❌ |
| 位置记忆 | 记住上次关闭位置 | ✅ |
| 全屏 | 支持 F11 切换 | ✅ |
