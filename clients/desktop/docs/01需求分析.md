## 1. 产品定位

Solra Desktop 是索拉平台的 **主力 3D 渲染客户端**，面向 Windows/macOS/Linux 桌面用户，承载以下核心定位：

- **3D 空间主入口**：提供完整的 GPU 加速 3D 渲染体验（Vulkan/OpenGL/Metal）
- **端侧 AI 推理**：集成 Core SDK 推理引擎（llama.cpp / ONNX Runtime）
- **实时社交**：WebRTC 音视频通话 + 空间内多人交互
- **高性能创作工具**：面向创作者的高帧率编辑预览环境

### 与其他客户端的关系

| 客户端 | 3D 渲染 | 端侧推理 | WebRTC | 定位 |
|--------|---------|---------|--------|------|
| **Desktop** | ✅ 主力 | ✅ 主力 | ✅ 主力 | 3D 空间 + 创作 |
| iOS | 🔶 辅助 | 🔶 辅助 | ✅ | 移动社交 |
| Android | 🔶 辅助 | 🔶 辅助 | ✅ | 移动社交 |
| Web | ❌ | ❌ | ❌ | 轻量信息浏览 |

## 3. 非功能需求

### 3.1 性能要求

| 指标 | 最低要求 | 推荐配置 |
|------|---------|---------|
| 帧率（3D 视口） | ≥ 30 FPS | ≥ 60 FPS |
| 帧率（UI 界面） | ≥ 60 FPS | ≥ 120 FPS |
| 冷启动时间 | < 5s | < 3s |
| GPU 显存占用 | < 2GB | < 1GB |
| 内存占用（空闲） | < 500MB | < 300MB |
| 推理延迟（LLM 首 Token） | < 3s | < 1s |

### 3.2 兼容性要求

| 平台 | 最低版本 | 推荐版本 |
|------|---------|---------|
| Windows | Windows 10 (1809+) | Windows 11 |
| macOS | macOS 12 (Monterey) | macOS 14 (Sonoma) |
| Linux | Ubuntu 22.04+ | Ubuntu 24.04 |

### 3.3 GPU 要求

| 等级 | 要求 | 说明 |
|------|------|------|
| 最低 | Vulkan 1.1 / OpenGL 4.1 / Metal 2 | 基础 PBR 渲染 |
| 推荐 | Vulkan 1.3 / DirectX 12 / Metal 3 | 完整效果 + 光线追踪 |
| 降级 | Software Rasterizer | 无 GPU 时的 CPU 软渲染 |

### 3.4 安装包要求

| 指标 | 目标 |
|------|------|
| Windows 安装包 | < 200MB（不含模型） |
| macOS 安装包 | < 250MB |
| Linux AppImage | < 200MB |
| 增量更新包 | < 50MB |

## 4. 架构约束

1. **Core SDK 强制依赖**：桌面客户端是唯一强制需要 Core SDK 的客户端，DLL 不可用时降级为有限功能模式
2. **原生 GPU 渲染**：3D 视口优先使用 Core SDK 原生渲染，Three.js WebGL 仅作为降级/调试模式
3. **本地推理优先**：端侧推理优先于云端推理，降低延迟、保护隐私
4. **离线可用**：基础功能（已缓存空间的浏览）需支持离线模式
