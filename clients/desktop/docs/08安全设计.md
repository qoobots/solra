## 2. 传输安全

### 2.1 通信加密

| 协议 | 加密方式 | 说明 |
|------|---------|------|
| REST API | HTTPS (TLS 1.3) | 所有 HTTP 请求强制加密 |
| WebSocket | WSS (TLS 1.3) | 实时通信加密 |
| gRPC | gRPC-TLS | 服务间调用加密 |
| WebRTC | DTLS-SRTP | 媒体流 + 数据通道加密 |

### 2.2 证书策略

| 策略 | 说明 |
|------|------|
| 证书验证 | 严格验证，拒绝自签名证书（开发环境除外） |
| 证书固定 | 生产环境启用 SSL Pinning |
| HSTS | 强制 HTTPS，不可降级 |

## 3. 认证与授权

### 3.1 认证流程

```
用户登录
  → POST /api/auth/login (HTTPS)
  → 服务器验证凭证
  → 返回 JWT Access Token (15min) + Refresh Token (7d)
  → 前端存储到 localStorage
  → 后续请求在 Authorization Header 携带 Bearer Token
  → Token 过期 → 使用 Refresh Token 刷新
  → Refresh Token 过期 → 重新登录
```

### 3.2 Token 管理

| 措施 | 说明 |
|------|------|
| Access Token 有效期 | 15 分钟 |
| Refresh Token 有效期 | 7 天 |
| Token 存储 | localStorage（前端）+ 内存（Rust 后端） |
| 退出登录 | 清除所有 Token + 调用服务端撤销接口 |
| 多设备登录 | 支持，各自独立 Token |

### 3.3 权限模型

| 角色 | 权限 |
|------|------|
| 访客 | 浏览公开空间 |
| 注册用户 | 进入空间、文字聊天、个人主页 |
| 创作者 | 创建/编辑空间、上传资产 |
| 管理员 | 审核内容、用户管理 |

## 4. 本地存储安全

### 4.1 数据隔离

| 数据 | 存储位置 | 加密 |
|------|---------|------|
| JWT Token | localStorage | ❌ (仅 HTTPS 传输) |
| 用户设置 | localStorage | ❌ |
| 消息历史 | IndexedDB | ❌ |
| 缓存资产 | IndexedDB / 文件系统 | ❌ (公开资产) |
| 用户模型文件 | 文件系统 | 🔶 可选加密 |

### 4.2 清理策略

| 触发条件 | 清理内容 |
|------|---------|
| 用户退出登录 | Token + 会话数据 |
| 应用卸载 | 所有本地数据 |
| 隐私模式 | 不持久化任何数据（未来） |

## 5. 运行时安全

### 5.1 Core SDK 安全

| 措施 | 说明 |
|------|------|
| DLL 签名验证 | 生产环境验证 libsolracore.dll 数字签名 |
| 哈希校验 | 加载前校验 DLL SHA256 |
| 隔离沙箱 | Core SDK 运行在独立线程，崩溃不影响主进程 |
| 内存保护 | Core SDK 内部分配的内存由 SDK 自身管理 |

### 5.2 WebView 安全

| 措施 | 说明 |
|------|------|
| CSP (Content Security Policy) | 限制脚本来源、禁止 eval() |
| 导航限制 | 仅允许加载白名单域名 |
| IPC 校验 | 后端验证所有 IPC 参数 |
| XSS 防护 | Vue 3 默认转义输出 |

### 5.3 CSP 策略

```
default-src 'self';
script-src 'self';
style-src 'self' 'unsafe-inline';
img-src 'self' https: data:;
connect-src 'self' https://api.solra.io wss://api.solra.io;
font-src 'self';
media-src 'self' blob:;
```

## 6. 代码安全

### 6.1 前端代码保护

| 措施 | 说明 |
|------|------|
| Vite 生产构建 | 代码压缩 + Tree Shaking |
| Sourcemap | 生产构建不包含 sourcemap |
| 调试检测 | 生产环境禁用 Vue Devtools |

### 6.2 Rust 后端保护

| 措施 | 说明 |
|------|------|
| Release 优化 | LTO + strip symbols |
| 反调试 | 检测调试器附加（未来） |
| 日志安全 | 生产环境不输出敏感信息 |

### 6.3 依赖安全

| 措施 | 说明 |
|------|------|
| npm audit | CI 中定期扫描前端依赖漏洞 |
| cargo audit | CI 中定期扫描 Rust 依赖漏洞 |
| 依赖锁定 | package-lock.json + Cargo.lock |
| SBOM | 生成软件物料清单 |

## 7. 数据最小化

| 原则 | 说明 |
|------|------|
| 仅收集必要数据 | 不收集设备唯一标识符 |
| 本地优先 | 能本地处理的不上传 |
| 用户可控 | 设置中可查看/导出/删除个人数据 |
| 匿名化 | 崩溃报告去标识化 |
