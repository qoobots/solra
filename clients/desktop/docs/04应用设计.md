## 1. 架构总览

### 1.1 五层架构

```
┌──────────────────────────────────────────────────────┐
│                    表现层 (UI)                         │
│  Vue 3 Components / Element Plus / SCSS              │
│  RenderViewport / SpaceList / Settings / Profile     │
├──────────────────────────────────────────────────────┤
│                   状态管理层                           │
│  Pinia Stores (render / space / user / settings)     │
│  Composables (useRenderer / useSpace / useAuth)      │
├──────────────────────────────────────────────────────┤
│                   服务层 (Service)                     │
│  SceneManager / SceneDataModel / OrbitController     │
│  ApiClient (axios) / IpcClient (Tauri invoke)        │
├──────────────────────────────────────────────────────┤
│               IPC 通信层 (Tauri Bridge)                │
│  render_cmd / space_cmd / avatar_cmd / system_cmd    │
│  core::render / core::ffi / core::inference          │
├──────────────────────────────────────────────────────┤
│                   引擎层 (Core SDK)                    │
│  libsolracore.dll/.so/.dylib (C++17)                 │
│  Render / Animation / Inference / Streaming / WebRTC │
└──────────────────────────────────────────────────────┘
```

### 1.2 数据流

```
用户输入 → Vue Component → Pinia Store → IPC invoke()
  → Rust IPC Handler → Core FFI → C++ Engine
  → 计算结果 → Rust → IPC Response → Pinia Store → Vue Reactivity → UI 更新
```

## 3. 核心流程

### 3.1 应用启动流程

```
1. main.rs → Tauri Builder 初始化
2. 注册所有 IPC 命令
3. 设置 Tauri 插件（shell/dialog/notification）
4. 启动 WebView → 加载 index.html
5. Vue 3 应用挂载 → 路由初始化
6. App.vue → AppLayout 渲染
7. 侧边栏 + 默认 3D 空间视图
8. RenderViewport.vue mounted:
   a. 调用 init_renderer IPC → Rust 加载 Core SDK DLL
   b. 若 DLL 存在 → Core SDK 渲染管线初始化
   c. 若 DLL 不存在 → Three.js WebGL 降级
   d. 创建 SceneManager → 加载 Demo 场景
   e. 启动 render loop（requestAnimationFrame）
```

### 3.2 渲染循环流程

```
requestAnimationFrame(renderLoop)
  ├── 若 Core SDK 可用:
  │   ├── invoke('render_frame') → Rust → Core SDK begin/end_frame
  │   ├── invoke('get_fps') → 获取帧率数据
  │   └── 更新 Pinia renderStore
  │
  └── 若 Three.js 降级:
      ├── OrbitController.update()
      ├── Three.js renderer.render(scene, camera)
      └── 更新 FPS 计数器
```

### 3.3 场景加载流程

```
用户选择空间 → invoke('load_scene', {sceneId})
  → Rust → Core SDK solra_scene_load()
  → 场景数据返回
  → SceneDataModel.parse(json)
  → SceneManager.createObjects(data)
  → Three.js 对象树构建
  → RenderViewport 更新显示
```

### 3.4 设置变更流程

```
用户在设置面板修改选项
  → Pinia settingsStore 更新
  → 持久化到 localStorage
  → 若涉及渲染设置:
    ├── invoke('update_render_config', config)
    └── 若后端需重建 → destroy_renderer + init_renderer
  → UI 即时响应
```

## 4. 状态管理设计

### 4.1 renderStore

```typescript
interface RenderState {
  isInitialized: boolean       // 渲染引擎是否初始化
  useCoreSdk: boolean          // 是否使用 Core SDK
  gpuBackend: string           // GPU 后端名称
  fps: number                  // 当前帧率
  frameTime: number            // 帧时间 (ms)
  resolution: { w: number; h: number }
  vsync: boolean
  msaa: number
}
```

### 4.2 spaceStore

```typescript
interface SpaceState {
  currentSpaceId: string | null
  spaces: SpaceInfo[]
  isLoading: boolean
  onlineUsers: number
  sceneData: SceneData | null
}
```

### 4.3 settingsStore

```typescript
interface SettingsState {
  // 图形
  renderBackend: 'auto' | 'vulkan' | 'opengl' | 'metal'
  resolutionScale: number       // 0.5 - 2.0
  fpsLimit: number              // 30 / 60 / 120 / unlimited
  vsync: boolean
  antiAliasing: 'off' | 'msaa2x' | 'msaa4x' | 'taa'
  shadowQuality: 'low' | 'medium' | 'high'
  // 音频
  audioInputDevice: string
  audioOutputDevice: string
  spatialAudio: boolean
  // 通用
  language: 'zh-CN' | 'en-US'
  theme: 'light' | 'dark' | 'auto'
}
```

## 5. 错误处理与降级

### 5.1 Core SDK 不可用

```
启动时 DLL 加载失败
  → ffi::is_core_sdk_loaded() = false
  → 所有 Core IPC 命令返回 Error
  → 前端检测 → useCoreSdk = false
  → 切换到 Three.js WebGL 渲染
  → GPU 信息显示 "WebGL (降级模式)"
  → 推理/WebRTC/流式加载功能不可用
```

### 5.2 GPU 后端降级

```
Vulkan 不可用 → 尝试 OpenGL 4.6
OpenGL 不可用 → 尝试 OpenGL ES 3.0
全部不可用 → Software Rasterizer
```

### 5.3 网络断开

```
HTTP 请求失败 → 本地缓存数据
WebSocket 断开 → 自动重连（指数退避）
空间数据不可用 → 显示已缓存的空间
```
