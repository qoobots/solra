# AI服务层（AI-Services）安全设计

> ai-services/ 安全——模型保护、推理内容安全、API鉴权

---

## 一、安全威胁模型

| 威胁 | 风险 | 缓解 |
|------|------|------|
| 提示注入 | 高 | 输入清洗+安全模型过滤 |
| 模型泄露 | 高 | 模型加密存储+访问控制 |
| API滥用 | 中 | Token鉴权+Rate Limit |
| GPU资源耗尽 | 中 | 并发限制+队列管理 |
| 推理数据泄露 | 中 | 推理日志脱敏 |

---

## 二、API鉴权

```
请求 → FastAPI中间件
  ├── API Key 验证 (服务间调用)
  ├── Rate Limit (令牌桶)
  └── 请求体大小限制 (10MB)
```

---

## 三、内容安全

```
用户输入 → safety-model-service 预检 → llm-router 推理 → safety-model-service 后检
```

---

## 四、模型保护

| 层级 | 措施 |
|------|------|
| 存储 | 模型文件 AES-256 加密 |
| 传输 | TLS 1.3 |
| 访问 | 仅推理进程可读取 |
| 审计 | 模型访问日志 |
