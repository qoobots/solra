# AI服务层（AI-Services）数据设计

> ai-services/ 数据管理——模型存储、推理缓存、指标数据

---

## 一、模型存储

```
models/
├── llm/
│   ├── qwen2.5-1.5b-q4/       # 端侧 (GGUF量化)
│   ├── qwen2.5-7b/            # 云端 (完整权重)
│   └── deepseek-v2-lite/      # 云端
├── safety/
│   └── solra-safety-v1/       # 安全分类器
├── embedding/
│   └── text2vec-large-chinese/
├── tts/
│   ├── vits-chinese/
│   └── cosyvoice/
└── registry/                   # MLflow 元数据
```

---

## 二、缓存策略

| 缓存对象 | TTL | 说明 |
|---------|-----|------|
| LLM KV Cache | 请求级 | 单次对话复用 |
| 嵌入向量 | 24h | Redis 缓存热门内容嵌入 |
| 安全分类结果 | 1h | 相同内容不重复审核 |
| TTS音频 | 永久 | 常用短语预合成缓存 |

## 三、指标数据

```python
# Prometheus 指标
inference_request_total{model,status}     # 推理请求计数
inference_duration_seconds{model}          # 推理延迟
gpu_memory_used_bytes{device}              # GPU显存
token_count_total{model}                   # Token计数
queue_depth{model}                         # 队列深度
```
