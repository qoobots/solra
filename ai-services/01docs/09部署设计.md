# AI服务层（AI-Services）部署设计

> ai-services/ 部署方案——Docker + GPU K8s节点 + 模型分发

---

## 一、容器化

```dockerfile
FROM nvidia/cuda:12.1-runtime-ubuntu22.04
RUN apt-get update && apt-get install -y python3.11
COPY dist/*.whl /app/
RUN pip install /app/*.whl
EXPOSE 8000
CMD ["uvicorn", "main:app", "--host", "0.0.0.0", "--port", "8000"]
```

---

## 二、GPU资源配置

| 服务 | GPU型号 | GPU数量 | 显存 |
|------|---------|---------|------|
| llm-router (7B) | A10 / 4090 | 1 | 24GB |
| llm-router (13B) | A100-40G | 1 | 40GB |
| safety-model | T4 / 4060 | 1 | 16GB |
| embedding | T4 | 1 | 16GB |
| tts | T4 | 1 | 16GB |

---

## 三、K8s部署

```yaml
resources:
  limits:
    nvidia.com/gpu: 1
nodeSelector:
    gpu-type: inference
```

---

## 四、模型分发

```
模型训练/更新
  → MLflow Registry
  → 模型打包 (Docker镜像层 / 对象存储挂载)
  → GPU节点预加载
  → 灰度切换
```
