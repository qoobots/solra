# AI服务层（AI-Services）工程目录结构

> ai-services/ 完整目录树与各模块职责

---

## 一、目录树

```
ai-services/
├── 01docs/                          # 本文档目录
│   ├── 01需求分析.md
│   ├── 02产品设计.md
│   ├── 03功能清单.md
│   ├── 04应用设计.md
│   ├── 05技术设计.md
│   ├── 06交互设计.md
│   ├── 07数据设计.md
│   ├── 08安全设计.md
│   ├── 09部署设计.md
│   ├── 10质量设计.md
│   ├── 11API设计.md
│   ├── 12工程目录.md                # 本文
│   └── 13开发进度.md
├── pyproject.toml                   # Poetry 根配置
├── shared/                          # 共享基础设施
│   ├── middleware/                  # 公共中间件
│   │   ├── auth.py                  # 鉴权
│   │   ├── logging.py               # 日志
│   │   └── rate_limit.py            # 限流
│   ├── scheduler/                   # GPU调度
│   ├── registry/                    # MLflow模型注册
│   └── metrics/                     # Prometheus指标
├── llm-router/                      # LLM路由 [P0]
│   ├── api/                         # FastAPI路由
│   ├── router/                      # 路由策略
│   └── adapters/                    # 模型适配器
├── safety-model-service/            # 安全模型 [P0]
├── embedding-service/               # 嵌入服务 [P1]
├── recommendation-pipeline/         # 推荐流水线 [P1]
├── tts-service/                     # 语音合成 [P1]
└── models/                          # 模型文件存储
    ├── llm/
    ├── safety/
    ├── embedding/
    └── tts/
```

---

## 二、模块职责

| 模块 | 优先级 | 职责 |
|------|--------|------|
| shared | P0 | 公共中间件/调度/注册/指标 |
| llm-router | P0 | LLM路由与流式输出 |
| safety-model-service | P0 | 文本/图片安全分类 |
| embedding-service | P1 | 文本/多模态向量化 |
| recommendation-pipeline | P1 | 协同过滤+内容推荐 |
| tts-service | P1 | 情感语音合成 |
| models/ | P0 | 模型文件版本管理 |
