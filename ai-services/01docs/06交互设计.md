# AI服务层（AI-Services）交互设计

> ai-services/ API交互——REST端点、SSE流式、gRPC调用

---

## 一、REST API

### 1.1 llm-router 端点

```
POST /v1/chat/completions    # 对话推理 (OpenAI兼容)
POST /v1/chat/stream         # SSE流式推理
GET  /v1/models               # 可用模型列表
```

### 1.2 safety-model-service 端点

```
POST /v1/safety/text          # 文本安全检测
POST /v1/safety/image         # 图片安全检测
POST /v1/safety/batch         # 批量审核
```

---

## 二、SSE 流式输出

```
GET /v1/chat/stream?text=你好
  → event: token
  → data: {"token":"你","prob":0.98}
  → event: token
  → data: {"token":"好","prob":0.95}
  → event: done
  → data: {"conversation_id":"xxx"}
```

---

## 三、统一响应格式

```json
{
    "code": 0,
    "message": "success",
    "data": { ... },
    "model_version": "qwen2.5-7b-v3",
    "inference_time_ms": 234
}
```

---

## 四、错误处理

| HTTP状态码 | 含义 |
|-----------|------|
| 200 | 成功 |
| 400 | 参数错误 |
| 429 | 推理队列满/限流 |
| 503 | 模型不可用/过载 |
