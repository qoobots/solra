# 开发工具链（Tools）交互设计

> tools/ 的"交互"——开发者通过命令行操作，工具输出清晰的反馈

---

## 一、CLI 交互规范

```bash
$ make dev-setup
[1/5] Installing Java 17... ✓
[2/5] Installing Python 3.11... ✓
[3/5] Installing C++ build tools... ✓
[4/5] Starting PostgreSQL... ✓
[5/5] Starting Redis... ✓
✅ Development environment ready!
```

---

## 二、错误处理

```bash
$ make dev-setup
[1/5] Installing Java 17... ✓
[2/5] Installing Python 3.11... ✗
  Error: Download failed. Check your network connection.
  Retry: make dev-setup --retry-from=2
```

---

## 三、帮助系统

```bash
$ make help
dev-setup    - Setup local development environment
proto-gen    - Generate code from Proto definitions
load-test    - Run load testing scenarios
release      - Create a new release
clean        - Clean all build artifacts
```
