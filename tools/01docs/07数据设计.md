# 开发工具链（Tools）数据设计

> tools/ 数据管理——配置文件、生成产物、压测结果

---

## 一、配置数据

```
tools/
├── dev-env/config.yaml       # 开发环境配置
│   ├── dependencies: [java, python, node, cmake]
│   ├── databases: [postgresql, redis, kafka]
│   └── ports: [8001-8010, 5432, 6379]
├── codegen/buf.gen.yaml.ref  # 代码生成配置参考
├── load-test/config.json     # 压测配置
└── release/version.yaml      # 版本管理
```

| 工具 | 输出 | 位置 |
