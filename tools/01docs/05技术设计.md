# 开发工具链（Tools）技术设计

> tools/ 技术选型——脚本语言与工具选择

---

## 一、技术选型

| 工具 | 语言 | 说明 |
|------|------|------|
| dev-env | Shell/PowerShell | 系统级操作 |
| codegen | buf CLI + Shell | Proto代码生成 |
| load-test | K6 (JavaScript) | 现代压测工具 |
| release | Shell/PowerShell | 版本发布 |
| scripts | PowerShell | Windows开发环境 |

---

## 二、跨平台策略

```
Windows:  *.ps1 脚本 (PowerShell 7+)
macOS/Linux: *.sh 脚本 (Bash 5+)
# 通过 Makefile 统一入口，内部判断平台调用对应脚本
```

---

## 三、Makefile 集成

```makefile
.PHONY: dev-setup proto-gen load-test release

dev-setup:
ifeq ($(OS),Windows_NT)
	powershell ./tools/dev-env/setup.ps1
else
	bash ./tools/dev-env/setup.sh
endif

proto-gen:
	buf generate

load-test:
	k6 run ./tools/load-test/scenarios/

release:
ifeq ($(OS),Windows_NT)
	powershell ./tools/release/release.ps1
else
	bash ./tools/release/release.sh
endif
```
