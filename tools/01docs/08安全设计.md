# 开发工具链（Tools）安全设计

> tools/ 安全——脚本安全、密钥保护、访问控制

---

## 一、安全原则

| 原则 | 说明 |
|------|------|
| 脚本可审计 | 所有脚本版本控制+Code Review |
| 无硬编码密钥 | 通过环境变量/Secret Manager |
| 最小权限 | 脚本所需权限显式声明 |
| 幂等安全 | 重复运行不造成损害 |

---

## 二、密钥管理

```
tools/release/
  ├── 需要: Docker Registry Token / GitHub Token
  ├── 来源: 环境变量 (GITHUB_TOKEN)
  └── 不提交: .env 文件加入 .gitignore

tools/dev-env/
  ├── 需要: 本地开发数据库密码
  └── 来源: 自动生成随机密码
```

---

## 三、安全检查

| 检查项 | 措施 |
|--------|------|
| 脚本权限 | 可执行脚本仅限必要范围 |
| 外部下载 | 校验SHA-256 |
| sudo使用 | 最小化+显式提示 |
| 敏感输出 | 不输出Token/密码到终端 |
