# 开发工具链（Tools）工程目录结构

> tools/ 完整目录树与各工具职责

---

## 一、目录树

```
tools/
├── 01docs/                          # 本文档目录
│   ├── 01需求分析.md
│   ├── 02产品设计.md
│   ├── 03功能清单.md
│   ├── 04应用设计.md
│   ├── 05技术设计.md
│   ├── 06交互设计.md
│   ├── 07数据设计.md
│   ├── 08安全设计.md
│   ├── 09部署设计.md
│   ├── 10质量设计.md
│   ├── 11API设计.md
│   ├── 12工程目录.md                # 本文
│   └── 13开发进度.md
├── dev-env/                         # 开发环境 [P0]
│   ├── setup.sh / setup.ps1         # 初始化脚本
│   ├── start.sh / start.ps1         # 启动脚本
│   ├── stop.sh / stop.ps1           # 停止脚本
│   └── config.yaml                  # 配置
├── codegen/                         # 代码生成 [P2]
│   ├── generate.sh / generate.ps1   # 代码生成
│   └── scaffold.sh / scaffold.ps1   # 服务脚手架
├── load-test/                       # 压测 [P3]
│   ├── scenarios/                   # 压测场景
│   │   ├── api-basic.js             # 基础API
│   │   ├── inference-load.js        # 推理压测
│   │   └── full-flow.js             # 全链路
│   ├── config.json                  # 压测配置
│   └── reports/                     # 报告输出
├── release/                         # 发布 [P3]
│   ├── release.sh / release.ps1     # 发布脚本
│   ├── changelog.sh                 # Changelog生成
│   └── version.yaml                 # 版本管理
├── scripts/                         # 通用脚本
│   ├── init-dirs.ps1                # 目录初始化 ✅
│   └── add-gitkeep.ps1              # GitKeep占位 ✅
├── ide/                             # IDE配置
├── db-migration/                    # 数据库迁移
└── Makefile                         # 顶层入口
```

---

## 二、工具职责

| 工具 | 优先级 | 职责 |
|------|--------|------|
| dev-env/ | P0 | 一键开发环境 |
| scripts/ | P0 | 通用辅助脚本 |
| codegen/ | P2 | Proto代码生成 |
| load-test/ | P3 | 性能压测 |
| release/ | P3 | 版本发布管理 |
