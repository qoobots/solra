# 开发工具链（Tools）产品设计

> tools/ 产品形态——面向开发者的自动化脚本与CLI工具

---

## 一、产品形态

tools/ 的"产品"是一组开发者可直接运行的脚本和工具：

```
开发者 → Makefile / 脚本调用 → tools/
  ├── dev-env/    → 一键启动环境
  ├── codegen/    → 代码生成
  ├── load-test/  → 性能压测
  └── release/    → 版本发布
```

---

## 二、使用方式

```bash
# 开发环境初始化
make dev-setup
# 或
./tools/dev-env/setup.sh

# Proto代码生成
make proto-gen
# 或
./tools/codegen/generate.sh

# 压测
make load-test
# 或
./tools/load-test/run.sh

# 发布
make release
# 或
./tools/release/release.sh
```

---

## 三、工具设计原则

| 原则 | 说明 |
|------|------|
| 幂等性 | 多次运行结果一致 |
| 跨平台 | Windows/Mac/Linux 均可运行 |
| 可中断 | Ctrl+C 安全退出 |
| 进度可见 | 启动/运行/完成/错误 有反馈 |
| 自文档 | --help 输出清晰说明 |
