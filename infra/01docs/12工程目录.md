## 二、模块职责

| 模块 | 优先级 | 职责 |
|------|--------|------|
| docker/ | P0 | 标准化容器镜像 |
| monitoring/ | P0 | 监控/日志/追踪/告警 |
| kubernetes/base/ | P0 | K8s基础资源 |
| terraform/ | P1 | 多云资源编排 |
| kubernetes/charts/ | P1 | 各服务Helm包 |
| kubernetes/argocd/ | P1 | GitOps自动部署 |
| apisix/ | P1 | API网关路由/认证/限流 |
