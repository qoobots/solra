# 基础设施层（Infra）安全设计

> infra/ 安全——K8s安全、网络安全、Secret管理、合规

---

## 一、安全层级

```
网络安全: VPC隔离 + 安全组 + NetworkPolicy
  → K8s安全: RBAC + PodSecurity + NetworkPolicy
    → 供应链安全: 镜像签名 + 漏洞扫描
      → Secret管理: 外部Secret + 加密存储
        → 审计: 操作审计日志
```

---

## 二、K8s 安全

| 措施 | 说明 |
|------|------|
| RBAC | 最小权限原则 |
| NetworkPolicy | 服务间网络隔离 |
| PodSecurity | 禁止特权容器 |
| ImagePolicy | 仅允许签名镜像 |
| Runtime Security | Falco 运行时检测 |

---

## 三、网络安全

```
Internet → DDoS防护 → WAF → APISIX
  → NetworkPolicy → 服务间mTLS
  → 安全组 → 仅允许必要端口
```

---

## 四、合规清单

| 检查项 | 要求 |
|------|------|
| 数据加密 | 传输TLS/存储AES |
| 访问审计 | 所有操作记录日志 |
| 最小权限 | RBAC最小权限 |
| 密钥轮换 | 定期轮换 |
| 漏洞扫描 | Trivy镜像扫描 |
