# 基础设施层（Infra）数据设计

> infra/ 数据管理——配置数据、Secret、状态存储

---

## 一、配置管理

| 配置类型 | 存储方式 | 示例 |
|---------|---------|------|
| 非敏感配置 | ConfigMap / Git | 端口号/超时时间 |
| 敏感配置 | External Secrets / Vault | 数据库密码/API Key |
| 环境差异 | Helm Values / Terraform Variables | 域名/实例规格 |

---

## 二、Secret 管理

```
External Secrets Operator
  → 从 Vault/AWS Secrets Manager 读取
  → 同步到 K8s Secret
  → 服务通过环境变量/文件挂载使用
```

---

## 三、状态存储

| 组件 | 状态存储 |
|------|---------|
| Terraform | S3/OSS 后端 (tfstate) |
| ArgoCD | K8s CRD + Git |
| Prometheus | TSDB (Persistent Volume) |
| Loki | 对象存储 (S3/OSS) |
