# 02_管理员操作手册

> **文档编号**：08-02 | **版本号**：v1.0 | **编制人**：产品组 | **状态**：已发布 | **更新日期**：2026-06-30

---

## 1. 管理后台入口

`https://admin.solra.io` （需管理员权限）

## 2. 用户管理

- 用户列表：搜索/筛选/查看详情
- 封禁/解封：用户详情 → 操作 → 封禁
- 角色管理：分配/撤销角色

## 3. 内容审核

- 审核队列：待审核空间/虚拟人/资源
- 审核操作：通过/驳回（附原因）
- 举报处理：查看举报 → 处置（警告/限制/封禁/驳回）

## 4. 系统配置

- 分类管理：空间分类/商品分类
- 推荐配置：热门空间置顶/推荐权重
- 公告管理：发布/编辑/下架系统公告

## 5. 数据看板

- DAU/MAU趋势
- 新增用户/留存率
- 收入数据/订单统计
- 内容审核统计

## 6. 常用CLI命令

```bash
./tools/admin-cli user list --status=BANNED
./tools/admin-cli user ban --userId=xxx --reason="违规内容"
./tools/admin-cli space review --spaceId=xxx --action=approve
```
