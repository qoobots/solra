## 1. 快速开始

### 注册与登录
1. 打开Solra App/Web → 点击"注册"
2. 输入手机号 → 获取验证码 → 设置密码
3. 或使用微信/Apple/Google一键登录

### 创建虚拟人
1. 首页点击"+" → "创建虚拟人"
2. 选择"从照片生成"或"手动创建"
3. 调整外观 → 确认 → 完成

### 探索空间
1. 首页浏览推荐空间卡片
2. 上下滑动切换 → 点击进入感兴趣的空间
3. 在空间内移动/聊天/互动

### AI对话
1. 点击虚拟人头像 → 进入对话
2. 输入文字或点击麦克风语音输入
3. 虚拟人会用文字+语音回复

## 2. 常见操作

| 操作 | 步骤 |
|------|------|
| 更换装扮 | 虚拟人详情 → 装扮 → 选择服装 |
| 添加好友 | 用户主页 → 添加好友 |
| 创建空间 | 创作者中心 → 创建空间 |
| 购买商品 | 商城 → 选择商品 → 购买 |

## 3. 移动端特有

- 横屏进入3D空间
- 双指缩放3D视图
- 长按空间卡片弹出菜单
