## 1. 通用字段字典

所有表必须包含的通用字段：

| 字段名 | 类型 | 长度/精度 | 可空 | 默认值 | 说明 |
|--------|------|-----------|------|--------|------|
| id | UUID | - | N | gen_random_uuid_v7() | 主键，UUID v7（时间有序） |
| created_at | TIMESTAMPTZ | - | N | NOW() | 创建时间（UTC） |
| updated_at | TIMESTAMPTZ | - | N | NOW() | 更新时间，自动更新 |
| version | INTEGER | - | N | 1 | 乐观锁版本号 |
| deleted_at | TIMESTAMPTZ | - | Y | NULL | 软删除标记 |
