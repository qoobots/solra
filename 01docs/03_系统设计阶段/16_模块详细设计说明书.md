## 2. M2 虚拟人交互模块详细设计

### 2.1 AI对话引擎详细设计

```
ConversationEngine:
  + chat(ConversationRequest): Flux<ConversationEvent>  // SSE流式
  + getHistory(conversationId, page): Page<Message>
  + deleteConversation(conversationId): void
  - detectIntent(text): Intent
  - routeModel(intent, context): ModelRoute
  - buildPrompt(context, memory): Prompt
  - callLLM(prompt): Flux<String>
  - callSLM(prompt): Flux<String>
  - updateMemory(conversationId, messages): void

IntentDetector:
  + detect(text): Intent
  // Intent: CHITCHAT / TASK / EMOTIONAL / RECOMMEND / QUESTION

ModelRouter:
  + route(intent, estimatedComplexity): ModelRoute
  // 简单闲聊 → SLM (设备端)
  // 复杂推理 → LLM (云端)
  // 推荐/搜索 → 专用API

MemoryService:
  + retrieve(avatarId, query, topK): List<Memory>
  + store(avatarId, content, importance): void
  + summarize(conversationId): String
  + decay(avatarId): void  // 低重要性记忆衰减
```

### 2.2 AI对话数据流

```
用户输入文本
    ↓
[IntentDetector] 意图识别
    ↓
[MemoryService] 检索相关记忆 (Milvus向量搜索)
    ↓
[ModelRouter] 路由决策
    ├── SLM路径: 设备端推理 (llama.cpp)
    │       ↓
    │   本地SLM推理
    │
    └── LLM路径: 云端推理 (vLLM)
            ↓
        [PromptBuilder] 构建Prompt (角色设定+记忆+上下文)
            ↓
        [LLMService] 调用vLLM (SSE流式返回)
            ↓
[EmotionEngine] 情感分析
    ↓
[MemoryService] 更新记忆
    ↓
返回: { text, emotion, tokens_used, latency_ms }
```

## 7. M8 内容安全模块详细设计

### 7.1 自动审核管道

```
Content → [TextFilter] → [ImageFilter] → [ModelFilter] → [AudioFilter] → Result

TextFilter:
  - AC自动机敏感词匹配 (<1ms)
  - NLP情感/意图分析
  - 违规文本分类模型

ImageFilter:
  - NSFW检测 (CNN, MobileNetV2)
  - OCR文字提取 + 敏感词过滤
  - 暴力/血腥检测

ModelFilter (3D):
  - 面数/顶点合规检查
  - 贴图NSFW检测
  - 骨骼结构合规

AudioFilter:
  - 语音转文字 + 敏感词
  - 声纹异常检测
```

### 7.2 审核决策矩阵

| 自动审核结果 | 置信度 | 处置 |
|-------------|--------|------|
| PASS | ≥0.95 | 自动通过 |
| PASS | 0.7-0.95 | 发布 + 随机抽检 |
| UNCERTAIN | 0.3-0.7 | 人工复审 |
| REJECT | 0.7-0.95 | 自动驳回 + 通知 |
| REJECT | ≥0.95 | 自动驳回 + 记录 |
