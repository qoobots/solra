## 1. 数据库总体设计

### 1.1 设计原则

| 原则 | 说明 |
|------|------|
| **每个微服务独立数据库** | 遵循数据主权原则 |
| **读写分离** | CQRS模式，命令/查询使用不同模型 |
| **多模态存储** | 关系型 + 文档型 + 向量型 + 对象存储混合 |
| **版本化管理** | Flyway管理数据库变更 |
| **命名规范** | 统一小写下划线命名 |

### 1.2 数据库清单

| 数据库 | 所属服务 | 引擎 | 用途 |
|--------|----------|------|------|
| auth_db | AUTH | PostgreSQL | 用户/角色/权限/会话 |
| avt_db | AVT | PostgreSQL | 虚拟人/对话/装扮 |
| avt_memory | AVT | Milvus | 虚拟人长期记忆向量 |
| spc_db | SPC | PostgreSQL | 空间/推荐/搜索 |
| spc_search | SPC | Elasticsearch | 空间全文搜索 |
| crt_db | CRT | PostgreSQL | 空间模板/资源 |
| soc_db | SOC | PostgreSQL | 好友/群组/消息 |
| grw_db | GRW | PostgreSQL | 任务/等级/成就/邀请 |
| mon_db | MON | PostgreSQL | 商品/订单/订阅/结算 |
| saf_db | SAF | PostgreSQL | 审核记录/举报/违规 |
| not_db | NOT | PostgreSQL | 通知/推送记录 |
| inf_db | INF | PostgreSQL | 配置/审计/健康检查 |

## 3. 核心表设计

### 3.1 通用字段规范

每个表必须包含以下字段：

| 字段 | 类型 | 说明 |
|------|------|------|
| id | UUID | 主键，使用UUID v7（时间有序） |
| created_at | TIMESTAMPTZ | 创建时间，默认 NOW() |
| updated_at | TIMESTAMPTZ | 更新时间，自动更新 |
| version | INTEGER | 乐观锁版本号 |
| deleted_at | TIMESTAMPTZ | 软删除标记（NULL=未删除） |

### 3.2 索引规范

| 规则 | 说明 |
|------|------|
| 主键索引 | 自动创建 |
| 外键索引 | 所有外键字段必须建索引 |
| 查询索引 | 根据查询模式创建复合索引 |
| 唯一索引 | 业务唯一性约束 |
| 全文索引 | 使用PostgreSQL GIN索引或ES |

### 3.3 分区策略

| 表 | 分区键 | 分区方式 | 分区粒度 |
|----|--------|----------|----------|
| messages | created_at | RANGE | 按月 |
| notifications | created_at | RANGE | 按月 |
| audit_logs | created_at | RANGE | 按月 |
| conversations | created_at | RANGE | 按月 |

## 4. 读写分离 (CQRS)

### 4.1 命令模型 (Write)

- 使用JPA Entity，完整聚合根
- 通过Repository进行增删改
- 发布领域事件

### 4.2 查询模型 (Read)

- 使用JDBC Template或jOOQ
- 针对查询场景优化的DTO
- 可跨表JOIN（查询侧放宽限制）
- 使用Redis缓存热点查询

### 4.3 同步策略

```
Write Service → 写主库 → 发布事件 → Kafka
                                      ↓
                              Read Service ← 消费事件 → 更新读模型
```

## 6. 迁移管理 (Flyway)

```
db/migration/
├── V1__init_auth_schema.sql
├── V2__add_oauth_tables.sql
├── V3__add_user_preferences.sql
└── ...
```

| 规则 | 说明 |
|------|------|
| 命名 | V{version}__{description}.sql |
| 版本号 | 递增整数，不可重复 |
| 不可变性 | 已执行的迁移不可修改 |
| 回滚 | 使用Undo迁移（生产慎用） |
| CI集成 | 应用启动时自动执行 |
