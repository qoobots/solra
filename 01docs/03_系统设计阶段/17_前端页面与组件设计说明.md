## 3. 组件设计规范

### 3.1 组件命名规范

```
格式: PascalCase，多单词
示例: SpaceCard, ChatPanel, UserAvatar

文件结构:
SpaceCard/
├── SpaceCard.vue          # 组件模板
├── SpaceCard.ts           # 组件逻辑 (Composition API)
├── SpaceCard.module.css   # CSS Modules
├── SpaceCard.test.ts      # 单元测试
└── index.ts               # 导出
```

### 3.2 组件通信规范

| 场景 | 方式 |
|------|------|
| 父子通信 | Props (父→子) + Emits (子→父) |
| 跨层级 | Provide/Inject |
| 全局状态 | Pinia Store |
| 事件总线 | mitt (仅限非Vue组件间) |
| 实时数据 | WebSocket + Pinia |

### 3.3 状态管理 (Pinia)

```
stores/
├── auth.store.ts          # 用户认证状态
├── space.store.ts         # 当前空间状态
├── chat.store.ts          # 对话状态
├── avatar.store.ts        # 虚拟人状态
├── notification.store.ts  # 通知状态
└── ui.store.ts            # UI状态（主题/语言/侧边栏）
```

## 4. 3D渲染组件 (Three.js)

### 4.1 架构设计

```
solra-3d/
├── core/
│   ├── SceneManager.ts    # 场景管理
│   ├── CameraController.ts # 相机控制（轨道/第一人称）
│   ├── LightManager.ts    # 光照管理
│   └── RenderLoop.ts      # 渲染循环
├── loaders/
│   ├── GLTFLoader.ts      # GLTF/GLB加载
│   ├── TextureLoader.ts   # 纹理加载（KTX2/Basis）
│   └── AvatarLoader.ts    # 虚拟人加载（含LOD）
├── objects/
│   ├── Avatar.ts          # 虚拟人3D对象
│   ├── SpaceObject.ts     # 空间物体
│   └── Skybox.ts          # 天空盒
├── effects/
│   ├── ParticleSystem.ts  # 粒子系统
│   ├── PostProcessing.ts  # 后处理（Bloom/DOF/SSAO）
│   └── ShadowSystem.ts    # 阴影系统
└── interaction/
    ├── Raycaster.ts       # 射线检测
    ├── DragController.ts  # 拖拽交互
    └── ClickHandler.ts    # 点击交互
```

### 4.2 性能优化

| 技术 | 说明 |
|------|------|
| LOD | 4级LOD自动切换 |
| Frustum Culling | 视锥体剔除 |
| Instance Rendering | 重复物体合批 |
| Texture Atlas | 纹理图集 |
| KTX2压缩 | GPU原生纹理压缩 |
| Web Worker | 物理计算放到Worker线程 |
| Object Pool | 粒子/特效对象池 |
