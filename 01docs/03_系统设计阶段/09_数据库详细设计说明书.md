## 1. AUTH数据库 (auth_db)

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | UUID | PK | 用户ID (UUID v7) |

| email | VARCHAR(255) | UNIQUE, 加密 | 邮箱 |
| password_hash | VARCHAR(255) | NOT NULL | bcrypt密码哈希 |
| nickname | VARCHAR(50) | NOT NULL | 昵称 |
| avatar_url | VARCHAR(500) | | 头像URL |
| status | VARCHAR(20) | NOT NULL | ACTIVE/INACTIVE/BANNED/DELETED |
| real_name_verified | BOOLEAN | DEFAULT FALSE | 是否实名认证 |
| youth_mode | BOOLEAN | DEFAULT FALSE | 青少年模式 |
| created_at | TIMESTAMPTZ | NOT NULL | |
| updated_at | TIMESTAMPTZ | NOT NULL | |
| version | INTEGER | NOT NULL | 乐观锁 |
| deleted_at | TIMESTAMPTZ | | 软删除 |

索引：`idx_users_phone`, `idx_users_email`, `idx_users_status`
