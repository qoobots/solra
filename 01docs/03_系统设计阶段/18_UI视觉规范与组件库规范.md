## 1. 设计系统概述

### 1.1 设计理念

**关键词**：未来感 | 沉浸式 | 轻量化 | 跨平台一致

- **未来感**：赛博朋克+极简融合，蓝紫渐变主色调
- **沉浸式**：3D空间内UI最小化干扰，半透明毛玻璃材质
- **轻量化**：Web/移动端快速加载，首屏<2秒
- **一致性**：Web/移动/桌面统一设计语言

### 1.2 设计工具链

| 工具 | 用途 |
|------|------|
| Figma | UI设计/原型/标注 |
| Design Tokens | 设计变量管理 |
| Storybook | 组件开发与文档 |

| Token | 色值 | 用途 |

### 2.4 渐变

```css
/* 品牌渐变（按钮/头部/高亮） */

/* 空间卡片渐变 */
--gradient-space: linear-gradient(180deg, transparent 40%, rgba(10,10,26,0.9));

/* 玻璃效果渐变 */
--gradient-glass: linear-gradient(135deg, rgba(255,255,255,0.1), rgba(255,255,255,0.05));
```

## 6. 动效规范

### 6.1 缓动函数

```css
--ease-out: cubic-bezier(0.16, 1, 0.3, 1);     /* 出现动画 */
--ease-in: cubic-bezier(0.4, 0, 1, 1);           /* 消失动画 */
--ease-in-out: cubic-bezier(0.65, 0, 0.35, 1);   /* 往返动画 */
--ease-spring: cubic-bezier(0.34, 1.56, 0.64, 1); /* 弹性动画 */
```

### 6.2 时长

| Token | 值 | 用途 |
|-------|-----|------|
| `--duration-fast` | 100ms | 微交互（hover/ripple） |
| `--duration-normal` | 200ms | 过渡（fade/toggle） |
| `--duration-slow` | 300ms | 页面转场 |
| `--duration-modal` | 400ms | 弹窗动画 |

## 7. 组件规范

### 7.1 按钮

| 变体 | 用途 | 样式 |
|------|------|------|
| Primary | 主操作 | 品牌渐变背景，白色文字 |
| Secondary | 次要操作 | 半透明边框，品牌色文字 |
| Ghost | 弱操作 | 无背景，悬停显示 |
| Danger | 危险操作 | 红色背景 |
| Icon | 图标按钮 | 纯图标，圆形 |

尺寸：`sm (32px)` | `md (40px)` | `lg (48px)`

### 7.2 输入框

```
状态: default → hover → focus → filled → error → disabled
样式: 深色背景 + 边框 + 聚焦发光效果
```

### 7.3 卡片

```
SpaceCard:
- 封面图（16:9）
- 标题 + 描述（最多2行）
- 创作者信息（头像+昵称）
- 统计信息（在线人数/点赞）
- 悬停放大效果 + 阴影
```

### 7.4 弹窗

```
类型: Dialog / Drawer / Popover / Tooltip
遮罩: rgba(0,0,0,0.6) + backdrop-filter blur(4px)
内容: 玻璃效果背景 + 圆角
```

### 7.5 加载

```
类型: Skeleton / Spinner / Progress Bar
策略: 骨架屏优先（感知性能优于转圈）
```

## 10. Design Tokens

```json
// tokens.json (跨平台共享)
{
  "color": {
    "primary": { "500": "#6C5CE7" }
  },
  "typography": {
    "fontFamily": { "base": "Inter, sans-serif" }
  },
  "spacing": { "4": "16px" }
}
```

通过 Style Dictionary 生成各平台变量：
- CSS Custom Properties → Web
- SCSS Variables → 备选
- JSON → 移动端
- Tailwind Config → 工具类
