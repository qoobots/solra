## 1. 统一报文规范

## 2. 错误码规范

### 2.1 错误码结构

```
格式: A B C D E  (5位数字)

A: 错误类别 (1位)
  1 = 系统错误
  2 = 业务错误
  3 = 第三方错误

B: 错误域 (1位)
  0 = 通用
  1 = AUTH
  2 = AVT
  3 = SPC
  4 = CRT
  5 = SOC
  6 = GRW
  7 = MON
  8 = SAF
  9 = NOT

CDE: 具体错误 (3位)
  001-099 = 参数错误
  100-199 = 认证/授权错误
  200-299 = 资源错误
  300-399 = 业务规则错误
  400-499 = 第三方错误
  500-599 = 系统内部错误
```

## 4. 多语言支持

请求头 `Accept-Language` 控制错误消息语言：

```
Accept-Language: zh-CN → 中文错误消息
Accept-Language: en-US → 英文错误消息
```

默认：zh-CN
