## 二、异常分类

| 异常类型 | 说明 | 是否重试 | 处理策略 |
|----------|------|----------|----------|
| 网络超时 | 请求未在超时时间内返回 | 是 | 指数退避重试 |
| 服务繁忙 | 服务端返回限流/过载 | 是 | 延迟重试 |
| 业务异常 | 业务逻辑校验失败 | 否 | 直接返回错误 |
| 数据异常 | 数据不一致/不存在 | 否 | 记录日志+告警 |
| 系统异常 | 服务端未知错误 | 是 | 有限次数重试 |
