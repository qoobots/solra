## 1. 拆分原则

### 1.1 DDD限界上下文驱动

每个微服务对应一个DDD限界上下文（Bounded Context），10个子项目 = 10个核心微服务域。

### 1.2 拆分原则

| 原则 | 说明 |
|------|------|
| **单一职责** | 每个服务只做一件事，做好一件事 |
| **高内聚低耦合** | 服务内部高度聚合，服务间松耦合 |
| **数据主权** | 每个服务拥有自己的数据存储 |
| **独立部署** | 每个服务可独立构建、测试、部署 |
| **异步优先** | 服务间通信优先使用事件驱动 |
| **容错隔离** | 服务故障不级联影响其他服务 |

## 3. 核心微服务详细设计

### 3.1 AUTH Service (qooauth)

| 属性 | 内容 |
|------|------|
| **所属子项目** | qooauth |
| **优先级** | P0 |
| **数据存储** | PostgreSQL (auth_db) |
| **缓存** | Redis (Session/Token) |
| **对外API** | REST (注册/登录/Token刷新/第三方OAuth) |
| **对内API** | gRPC (权限验证/用户信息查询) |
| **消费事件** | 无 |
| **发布事件** | UserRegistered, UserLoggedIn, UserDeleted |

**聚合根**：
- User (用户ID/手机号/邮箱/密码哈希/状态/注册时间)
- Session (会话ID/用户ID/Token/过期时间/设备信息)
- Role (角色ID/名称/权限列表)

### 3.2 AVT Service (qoobrain + qoobody)

| 属性 | 内容 |
|------|------|
| **所属子项目** | qoobrain (AI对话) + qoobody (3D虚拟人) |
| **优先级** | P0 |
| **数据存储** | PostgreSQL (avt_db) + Milvus (向量记忆) |
| **缓存** | Redis (对话上下文/热点角色) |
| **对外API** | REST (创建/装扮/对话) + WebSocket (实时对话流) |
| **对内API** | gRPC (虚拟人信息查询/批量推理请求) |
| **消费事件** | UserRegistered (创建默认虚拟人) |
| **发布事件** | AvatarCreated, AvatarUpdated, ConversationCompleted |

**聚合根**：
- Avatar (虚拟人ID/用户ID/外观参数/风格/状态)
- Conversation (会话ID/虚拟人ID/用户ID/上下文/记忆引用)
- Memory (记忆ID/虚拟人ID/内容/向量/重要性/时间)

**子模块**：
- `avt-brain`：AI对话推理、意图识别、记忆管理
- `avt-body`：3D形象管理、动作驱动、表情生成
- `avt-voice`：ASR语音识别、TTS语音合成

### 3.3 SPC Service (qooweb + qoocloud)

| 属性 | 内容 |
|------|------|
| **所属子项目** | qooweb (Web门户) + qoocloud (云服务) |
| **优先级** | P0 |
| **数据存储** | PostgreSQL (spc_db) + Elasticsearch (搜索) |
| **缓存** | Redis (热门空间/推荐缓存) |
| **对外API** | REST (空间浏览/搜索/推荐/进入) + WebSocket (空间内实时交互) |
| **对内API** | gRPC (空间信息查询/批量推荐) |
| **消费事件** | SpaceCreated, SpaceUpdated, UserEnteredSpace |
| **发布事件** | UserEnteredSpace, UserLeftSpace, SpaceInteraction |

**聚合根**：
- Space (空间ID/创建者ID/名称/描述/分类/标签/容量/权限/热度)
- SpaceInstance (实例ID/空间ID/当前人数/节点/状态)
- Recommendation (推荐ID/用户ID/空间ID列表/算法版本/生成时间)

**子模块**：
- `spc-discovery`：空间发现/搜索/推荐
- `spc-session`：空间会话管理/负载均衡
- `spc-realtime`：空间内实时通信（WebSocket/WebRTC信令）
- `spc-live`：直播/活动管理

### 3.4 CRT Service (qoocore + qoodev)

| 属性 | 内容 |
|------|------|
| **所属子项目** | qoocore (核心引擎) + qoodev (开发者工具) |
| **优先级** | P0 |
| **数据存储** | PostgreSQL (crt_db) + MinIO (资源文件) |
| **缓存** | Redis (编辑会话/预览) |
| **对外API** | REST (空间创建/编辑/发布) |
| **对内API** | gRPC (空间模板查询/资源引用) |
| **消费事件** | 无 |
| **发布事件** | SpaceCreated, SpaceUpdated, SpacePublished, AssetUploaded |

**聚合根**：
- SpaceTemplate (模板ID/创建者ID/场景图/配置/版本)
- Asset (资源ID/上传者ID/类型/URL/元数据/审核状态)

**子模块**：
- `crt-editor`：空间编辑API
- `crt-asset`：资源管理/上传/优化
- `crt-template`：模板市场

### 3.5 SOC Service (qoosvc)

| 属性 | 内容 |
|------|------|
| **所属子项目** | qoosvc |
| **优先级** | P1 |
| **数据存储** | PostgreSQL (soc_db) + Redis (在线状态) |
| **缓存** | Redis (好友列表/状态) |
| **对外API** | REST (好友/群组/消息) + WebSocket (实时消息) |
| **对内API** | gRPC (好友关系查询/在线状态) |
| **消费事件** | UserRegistered, UserEnteredSpace, UserLeftSpace |
| **发布事件** | FriendshipEstablished, MessageSent, PresenceChanged |

**聚合根**：
- Friendship (关系ID/用户A/用户B/状态/建立时间)
- Group (群组ID/创建者/名称/成员列表/设置)
- Message (消息ID/发送者/接收者/类型/内容/时间/状态)

### 3.6 GRW Service (qoochain)

| 属性 | 内容 |
|------|------|
| **所属子项目** | qoochain |
| **优先级** | P1 |
| **数据存储** | PostgreSQL (grw_db) + Redis (任务进度) |
| **对外API** | REST (任务/等级/成就/邀请) |
| **对内API** | gRPC (用户等级查询/任务状态) |
| **消费事件** | UserRegistered, UserLoggedIn, SpaceInteraction, PurchaseCompleted |
| **发布事件** | TaskCompleted, LevelUp, AchievementUnlocked |

### 3.7 MON Service (qoostore)

| 属性 | 内容 |
|------|------|
| **所属子项目** | qoostore |
| **优先级** | P2 |
| **数据存储** | PostgreSQL (mon_db) |
| **对外API** | REST (商品浏览/购买/订单/订阅/创作者中心) |
| **对内API** | gRPC (资产查询/余额查询) |
| **消费事件** | UserRegistered, AssetUploaded, SpacePublished |
| **发布事件** | PurchaseCompleted, SubscriptionChanged, RevenueSettled |

### 3.8 SAF Service (qoocompliance)

| 属性 | 内容 |
|------|------|
| **所属子项目** | qoocompliance |
| **优先级** | P1 |
| **数据存储** | PostgreSQL (saf_db) |
| **对外API** | REST (举报/审核结果查询) |
| **对内API** | gRPC (内容检查/敏感词过滤) |
| **消费事件** | SpacePublished, AssetUploaded, MessageSent, AvatarCreated |
| **发布事件** | ContentApproved, ContentRejected, UserReported |

### 3.9 NOT Service (qoogear)

| 属性 | 内容 |
|------|------|
| **所属子项目** | qoogear |
| **优先级** | P3 |
| **数据存储** | PostgreSQL (not_db) |

| **对内API** | gRPC (发送通知) |
| **消费事件** | 几乎所有业务事件 |
| **发布事件** | NotificationSent, NotificationRead |

### 3.10 INF Service (qoocommunity)

| 属性 | 内容 |
|------|------|
| **所属子项目** | qoocommunity |
| **优先级** | P3 |
| **数据存储** | PostgreSQL (inf_db) |
| **对外API** | REST (配置查询/健康检查) |
| **对内API** | gRPC (配置推送/健康上报) |
| **消费事件** | 所有服务注册/注销事件 |
| **发布事件** | ConfigChanged, ServiceHealthChanged |

## 4. 服务间通信矩阵

|  | AUTH | AVT | SPC | CRT | SOC | GRW | MON | SAF | NOT | INF |
|--|------|-----|-----|-----|-----|-----|-----|-----|-----|-----|
| **AUTH** | - | E | E | E | E | E | E | - | - | - |
| **AVT** | R | - | - | - | - | - | - | - | - | - |
| **SPC** | R | R | - | R | R | - | - | - | - | - |
| **CRT** | R | - | - | - | - | - | E | E | - | - |
| **SOC** | R | - | R | - | - | - | - | - | - | - |
| **GRW** | R | - | E | - | - | - | E | - | - | - |
| **MON** | R | - | - | R | - | - | - | - | - | - |
| **SAF** | - | E | E | E | E | - | - | - | - | - |
| **NOT** | E | E | E | E | E | E | E | E | - | - |
| **INF** | R | R | R | R | R | R | R | R | R | - |

> R = 同步调用(gRPC/REST)，E = 异步事件(Kafka)

## 5. 服务边界规则

| 规则 | 说明 |
|------|------|
| 禁止跨服务数据库访问 | 每个服务只能访问自己的数据库 |
| 禁止同步调用链>3层 | 超过3层必须改为异步事件 |
| 事件至少一次投递 | Kafka + 幂等消费保证 |
| 服务发现 | Consul/Nacos + K8s Service |
| 配置管理 | Nacos配置中心，支持热更新 |
