## 三、数据分级分类

| 数据级别 | 级别定义 | 数据示例 | 安全要求 |
|----------|----------|----------|----------|
| L1 公开 | 可公开信息 |          | 无需特殊保护 |
| L2 内部 | 内部使用 |            | 访问控制 |
| L3 敏感 | 敏感信息 |            | 加密+审计 |
| L4 机密 | 核心机密 |            | 最高级别保护 |

## 四、数据脱敏方案

### 4.1 脱敏场景
| 场景 | 脱敏要求 | 脱敏方式 |
|------|----------|----------|
| 开发测试 | 不可识别真实数据 | 静态脱敏 |
| 数据展示 | 部分信息隐藏 | 动态脱敏 |
| 数据导出 | 按权限脱敏 | 动态脱敏 |
| 日志记录 | 敏感信息屏蔽 | 动态脱敏 |

### 4.2 脱敏规则

| 数据类型 | 脱敏规则 | 示例 |
|----------|----------|------|
| 姓名 | 保留首字，其余* | 张** |
| 身份证号 | 保留前3后4，其余* | 310***********1234 |
| 手机号 | 保留前3后4，其余* | 138****5678 |
| 银行卡号 | 保留后4位，其余* | ***********1234 |
| 邮箱 | 保留首字母和域名 | t***@example.com |
