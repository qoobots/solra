## 1. 密码应用总览

| 场景 | 算法 | 密钥长度 | 标准 |
|------|------|----------|------|
| 传输加密 | TLS 1.3 (ECDHE-RSA-AES256-GCM) | 256位 | GB/T 38636 |
| 数据存储加密 | AES-256-GCM | 256位 | GB/T 32918 |
| 密码哈希 | bcrypt (cost=12) | - | - |
| 数字签名 | RS256 (JWT) | 2048位 | GB/T 32918 |
| 密钥交换 | ECDHE | 256位 | GB/T 38636 |
| 哈希校验 | SHA-256 | 256位 | GB/T 32905 |

## 2. 密钥管理

| 密钥类型 | 存储 | 轮换周期 |
|----------|------|----------|
| TLS私钥 | Vault | 90天 |
| JWT签名密钥 | Vault | 90天 |
| 数据库密码 | Vault | 30天 |
| 数据加密密钥(DEK) | Vault | 180天 |
| 密钥加密密钥(KEK) | HSM/Vault | 365天 |

## 3. 密码合规

- 符合GM/T 0054密码应用基本要求
- 使用国家密码管理局批准的算法
- 密钥全生命周期管理
