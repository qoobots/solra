## 1. 端到端场景

| 编号 | 场景 | 步骤 | 覆盖服务 |
|------|------|------|----------|
| E2E-01 | 新用户注册并创建虚拟人 | 注册→登录→创建虚拟人→AI对话 | AUTH/AVT |
| E2E-02 | 浏览空间并进入互动 | 浏览推荐→搜索→进入空间→聊天→离开 | SPC/SOC |
| E2E-03 | 创作者发布空间 | 登录→创建空间→编辑→上传资源→发布→审核 | CRT/SAF |
| E2E-04 | 购买虚拟商品 | 浏览商城→下单→支付→获得商品→装备 | MON/AVT |
| E2E-05 | 用户举报处理 | 举报→自动审核→人工审核→处置→通知 | SAF/NOT |

## 2. E2E-01 详细步骤

```
1. 打开App → 点击注册
2. 输入手机号 → 获取验证码 → 设置密码
3. 注册成功 → 自动登录 → 进入首页
4. 点击"创建虚拟人" → 选择"从照片生成"
5. 上传照片 → 等待AI生成 → 确认
6. 进入AI对话 → 发送"你好"
7. 验证收到回复（文本+表情）
```
