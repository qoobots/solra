## 1. 数据分类

| 类型 | 说明 | 来源 |
|------|------|------|
| 基础数据 | 角色/分类/配置 | SQL种子脚本 |
| 测试数据 | 用户/空间/虚拟人 | Faker生成 |
| 场景数据 | 特定业务流程 | 手工构造 |
| 性能数据 | 大数据量压测 | 批量生成脚本 |

## 2. 数据生成工具

```python
# tools/generate_test_data.py
# 生成1000用户 + 500虚拟人 + 200空间 + 10000消息

python tools/generate_test_data.py \
  --users 1000 \
  --avatars 500 \
  --spaces 200 \
  --messages 10000 \
  --output fixtures/
```

## 3. 数据隔离

- 每个测试类使用独立数据，通过`@Transactional`回滚
- 并行测试使用不同数据分区（用户ID范围隔离）
- 敏感字段脱敏：手机号→1380000xxxx

## 4. 性能测试数据量
