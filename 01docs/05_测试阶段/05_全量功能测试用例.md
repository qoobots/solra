## 2. 核心用例示例

### TC-AUTH-001: 手机号注册

| 字段 | 内容 |
|------|------|
| 前置条件 | 手机号未注册 |
| 步骤 | 1.输入手机号→2.获取验证码→3.输入验证码+密码→4.提交 |
| 预期 | 201, 返回Token+用户信息 |

### TC-AUTH-002: 重复注册

| 字段 | 内容 |
|------|------|
| 前置条件 | 手机号已注册 |
| 步骤 | 使用已注册手机号注册 |
| 预期 | 409, 错误码10105 |

### TC-AVT-001: AI创建虚拟人

| 字段 | 内容 |
|------|------|
| 前置条件 | 已登录, 上传正面照片 |
| 步骤 | 1.选择"从照片生成"→2.上传照片→3.等待AI生成→4.确认 |
| 预期 | 201, 返回虚拟人信息（含模型URL） |

### TC-AVT-002: AI对话

| 字段 | 内容 |
|------|------|
| 前置条件 | 已创建虚拟人 |
| 步骤 | 1.进入对话→2.输入"你好"→3.等待回复 |
| 预期 | 流式返回文本, 包含情感标签 |

### TC-SPC-001: 进入公开空间

| 字段 | 内容 |
|------|------|
| 前置条件 | 已登录, 空间未满 |
| 步骤 | 1.点击空间卡片→2.点击"进入" |
| 预期 | 返回WebSocket地址, 进入3D空间 |

### TC-SPC-002: 空间已满

| 字段 | 内容 |
|------|------|
| 前置条件 | 空间人数=容量上限 |
| 步骤 | 尝试进入 |
| 预期 | 403, 错误码10302 |

### TC-MON-001: 购买商品

| 字段 | 内容 |
|------|------|
| 前置条件 | 余额充足 |
| 步骤 | 1.选择商品→2.确认购买→3.支付 |
| 预期 | 200, 商品发放到账户 |
