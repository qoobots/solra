## 一、文档概述

### 1.1 编写目的

本文档定义索拉（Solra）项目的业务需求，描述业务背景、业务目标、核心业务能力和业务约束，作为后续产品需求（PRD）和技术方案设计的输入依据。

### 1.2 适用范围

适用于索拉项目全生命周期——10 个子项目、78 个功能、4 个客户端平台。

### 1.3 术语与定义

| 术语 | 定义 |
|------|------|
| 空间 | 3D 可交互虚拟场景，包含环境、虚拟人、事件系统的完整聚合 |
| 虚拟人 | 由 LLM 驱动、具备对话/情感/记忆的 AI 角色 |
| 刷空间 | 用户连续进入推荐空间消费内容的行为流 |
| 存在值 | 用户在索拉中"存在深度"的量化积分 |
| 决定性时刻 | 用户成长漏斗中触发不可逆转化的关键体验锚点 |

## 二、业务背景与目标

### 2.1 业务背景

陌生人社交产品遵循"网络带宽驱动内容形态升级"的历史规律：
- 3G 时代（2008）：图文社交，代表产品微博
- 4G 时代（2013-2015）：短视频社交，代表产品抖音
- 5G 时代（2022-至今）：3D 空间社交，代表产品索拉

当前 5G 渗透率处于爬坡期（类似 2014 年 4G），2 秒时间窗口内可下载约 200MB 数据，使 200-900MB 的 3D 空间实时加载成为可能。市场上缺乏将"AI 虚拟人 + 3D 空间 + 算法推荐"三者融合的产品，存在明确的市场空白。

### 2.2 业务目标

| 目标 | 度量指标 | 达成时间 |
|------|---------|---------|
| 验证核心体验闭环 | 空间进入 < 2s + 虚拟人互动 | 技术验证期（第 1-2 年） |
| 建立用户留存驱动 | 次日留存 > 40% | 产品验证期（第 3-4 年） |
| 实现规模化增长 | DAU > 1000 万 | 规模验证期（第 5-7 年） |
| 商业化变现 | 付费转化率 > 10%，年收入 > ¥10 亿 | 规模验证期 |

## 四、核心业务需求

### 4.1 业务功能需求

#### 4.1.1 空间消费

| 需求编号 | 需求描述 | 优先级 |
|---------|---------|--------|
| BR-SPC-001 | 空间流式加载，200MB 空间在 2 秒内进入可交互状态 | P0 |
| BR-SPC-002 | 个性化推荐流，首页展示推荐空间卡片 | P0 |
| BR-SPC-003 | 空间搜索与分类浏览 | P1 |
| BR-SPC-004 | 空间排行榜（日/周/月热门） | P2 |
| BR-SPC-005 | 退出空间时展示下一个空间 3D 预览 | P0 |

#### 4.1.2 虚拟人交互

| 需求编号 | 需求描述 | 优先级 |
|---------|---------|--------|
| BR-AVT-001 | 虚拟人实时对话（语音/文字双模态） | P0 |
| BR-AVT-002 | 虚拟人社交主动性（进入感知+主动招呼） | P0 |
| BR-AVT-003 | 虚拟人长期记忆（跨会话个性化回应） | P1 |
| BR-AVT-004 | 虚拟人 5 维情感模型 | P1 |
| BR-AVT-005 | 虚拟人个性化训练 | P2 |

### 4.2 业务流程需求

**核心业务流程："刷空间"循环**

```
进入空间 → 虚拟人互动 → 浏览空间 → 退出空间 → 预览下一空间 → 再次进入
```

**信仰转化漏斗**：游客 → 使用者 → 常客 → 信徒 → 布道者

### 4.3 业务规则需求

| 规则编号 | 规则描述 |
|---------|---------|
| BR-RL-001 | 免费用户每天最多 100 次空间切换 |
| BR-RL-002 | 虚拟人对话需经安全过滤器，违规内容自动拦截 |
| BR-RL-003 | 空间发布需经 AI+人工双重审核 |
| BR-RL-004 | 未成年人每日使用时长限制 2 小时，22:00-6:00 禁用 |
| BR-RL-005 | 创作者交易平台抽成 30%，月结，最低提现 ¥100 |

## 五、业务约束条件

| 约束 | 说明 |
|------|------|
| 网络环境 | 5G/WiFi 优先，4G 降级体验 |
| 设备要求 | iOS 17+ / Android 12+ / 中高端 GPU |
| 合规要求 | 等保三级 + 个人信息保护法 + 未成年人保护 |
| 数据存储 | 中国用户数据 100% 境内存储 |
| 内容安全 | 所有 UGC 内容需审核通过方可发布 |

## 六、业务验收标准

| 验收项 | 标准 |
|--------|------|
| 空间加载 | P95 < 2 秒（5G/WiFi 环境） |
| 虚拟人对话 | 端侧 < 1s 首 Token，云端 < 3s P99 |
| 推荐流 | CTR > 15%，用户平均刷空间深度 > 5 个/次 |
| 内容安全 | 漏审率 < 0.01%，举报处理 < 4 小时 |
| 可用性 | 核心服务 99.9%+ |
