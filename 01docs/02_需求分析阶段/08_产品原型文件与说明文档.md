## 2. 核心页面原型说明

### 2.1 首页推荐流

**设计目标**：让用户在3秒内找到感兴趣的空间或虚拟人。

**布局结构**：
```
┌─────────────────────────────┐
│  [搜索框]         [通知🔔] [头像] │  ← 顶部导航栏
├─────────────────────────────┤
│  推荐 | 热门 | 关注 | 附近   │  ← 分类Tab
├─────────────────────────────┤
│  ┌───────────────────────┐  │
│  │                       │  │
│  │    沉浸式空间卡片      │  │  ← 全屏卡片流
│  │    (封面图+标题+人数)  │  │
│  │                       │  │
│  └───────────────────────┘  │
│  ┌───────────────────────┐  │
│  │                       │  │
│  │    下一个空间卡片      │  │
│  │                       │  │
│  └───────────────────────┘  │
├─────────────────────────────┤
│  [首页]  [探索]  [+]  [消息] [我] │  ← 底部导航
└─────────────────────────────┘
```

**交互说明**：
- 上下滑动切换空间卡片
- 左滑查看空间详情
- 右滑进入空间
- 双击点赞
- 长按弹出快捷菜单

### 2.2 3D空间内HUD

**设计目标**：在沉浸式3D体验中提供轻量、不打扰的UI层。

**布局结构**：
```
┌─────────────────────────────┐
│ [←离开]  空间名称   [#在线人数] │  ← 顶部状态栏（半透明）
├─────────────────────────────┤
│                             │
│                             │
│         3D 空间主视图        │
│                             │
│                             │
│                    ┌──────┐ │
│                    │ 小地图 │ │  ← 右下角小地图
│                    └──────┘ │
│              ┌──────────┐  │
│              │ 🎤 💬 ✋ ⚙ │  │  ← 底部操作栏
│              └──────────┘  │
│  ┌─────────────────────┐   │
│  │  聊天消息浮层         │   │  ← 可收起
│  └─────────────────────┘   │
└─────────────────────────────┘
```

### 2.3 虚拟人创建向导

**设计目标**：引导用户5分钟内创建个性化虚拟形象。

**步骤流程**：
1. **选择基础**：性别 → 基础脸型 → 肤色
2. **面部定制**：眼睛/鼻子/嘴巴/耳朵/发型（滑块+预设）
3. **体型调整**：身高/体重/肩宽/比例
4. **风格选择**：写实/二次元/赛博朋克/幻想
5. **预览确认**：360°旋转预览 → 命名 → 完成

│        │ 有趣的空间给我吧  │ │
│        └─────────────────┘ │
│  ┌───────────────────────┐  │
│  │ 虚拟人：我推荐"星空    │  │

## 3. 交互设计规范

### 3.1 通用交互原则

| 原则 | 说明 |
|------|------|
| **即时反馈** | 所有操作200ms内给出视觉反馈 |
| **容错设计** | 关键操作可撤销（5秒内），删除需确认 |
| **渐进披露** | 核心功能一键可达，高级功能二级菜单 |
| **一致性** | 同平台内相同操作保持一致交互模式 |
| **无障碍** | 支持键盘导航、屏幕阅读器、高对比度模式 |

### 3.2 手势定义（移动端）

| 手势 | 操作 | 场景 |
|------|------|------|
| 单指滑动 | 浏览/滚动 | 列表浏览 |
| 单指双击 | 点赞 | 空间卡片 |
| 双指缩放 | 缩放视图 | 3D空间/虚拟人预览 |
| 单指长按 | 快捷菜单 | 空间卡片/消息 |
| 左滑 | 查看详情 | 空间卡片 |
| 右滑 | 进入空间 | 空间卡片 |

### 3.3 过渡动画规范

| 转场类型 | 时长 | 缓动函数 | 用途 |
|----------|------|----------|------|
| 页面切换 | 300ms | ease-in-out | 导航切换 |
| 弹窗出现 | 200ms | ease-out | 对话框/菜单 |
| 列表加载 | 150ms/item | ease-out | 瀑布流加载 |
| 微交互 | 100ms | ease | 按钮按下/悬停 |
| 加载动画 | 循环 | linear | 数据加载中 |
