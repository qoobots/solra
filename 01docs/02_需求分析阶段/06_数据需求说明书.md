## 一、数据存储需求总览

| 存储类型 | 技术选型 | 用途 | 规模预估（成熟期） |
|----------|---------|------|-----------------|
| 关系数据库 | PostgreSQL 16 | 用户/订阅/订单/会话等结构化数据 | 100TB+ |
| 缓存 | Redis 7 (Cluster) | 会话状态/推荐流/排行榜/情感状态 | 5TB |
| 向量数据库 | Milvus 2.x | 虚拟人记忆/空间特征向量 | 50TB |
| 搜索引擎 | Elasticsearch 8 | 空间搜索/审核日志 | 10TB |
| 对象存储 | MinIO / S3 | 空间资产/用户上传/模型文件 | 5PB |
| 消息队列 | Kafka | 领域事件/行为日志 | 1000 万 msg/s |
| 时序数据库 | Prometheus/VictoriaMetrics | 可观测性指标 | PB 级 |

## 二、核心数据域

### 2.1 用户数据

| 数据项 | 类型 | 说明 | 敏感级别 |
|--------|------|------|---------|
| user_id | BIGINT | 用户唯一标识 | 内部 |
| phone_number | VARCHAR(20) | 手机号（加密存储） | 敏感 |
| nickname | VARCHAR(50) | 昵称 | 公开 |
| presence_score | BIGINT | 存在值 | 公开 |
| faith_level | TINYINT | 信仰等级 1-18 | 内部 |
| subscription_tier | ENUM | 订阅档位 | 内部 |
| real_name | VARCHAR(50) | 实名（加密存储） | 机密 |

### 2.2 空间数据

| 数据项 | 类型 | 说明 |
|--------|------|------|
| space_id | VARCHAR(64) | 空间唯一标识 |
| owner_id | BIGINT | 创建者 |
| title | VARCHAR(100) | 空间名称 |
| category | ENUM | 分类 |
| status | ENUM | 状态（草稿/审核中/已发布/已下架） |
| asset_size | BIGINT | 资产总大小 |
| visit_count | BIGINT | 累计访问数 |
| avg_duration | INT | 平均停留秒数 |

### 2.3 虚拟人数据

| 数据项 | 类型 | 说明 |
|--------|------|------|
| avatar_id | VARCHAR(64) | 虚拟人唯一标识 |
| personality_template | ENUM | 性格模板 |
| emotion_state | JSON | 5 维情感向量 |
| affinity_level | TINYINT | 好感度 1-10 |
| memory_count | INT | 记忆点数量 |

## 三、数据指标需求

### 3.1 核心业务指标

| 指标 | 口径 | 用途 |
|------|------|------|
| DAU | 日活跃用户数 | 增长监控 |
| 空间进入率 | 点击→进入转化 | 推荐效果 |
| 平均停留时长 | 空间内停留秒数 | 内容质量 |
| 虚拟人互动率 | 进入后对话比例 | 核心体验 |
| 次日留存 | D0 活跃用户 D1 回访比例 | 留存驱动 |
| 付费转化率 | 付费用户 / DAU | 商业化 |
| ARPPU | 付费用户月均消费 | 收入预测 |

### 3.2 推荐效果指标

| 指标 | 口径 |
|------|------|
| CTR | 推荐点击率 |
| 刷空间深度 | 单次平均浏览空间数 |
| 跳过率 | 空间 < 3 秒退出比例 |
| 收藏率 | 空间收藏 / 进入 |

## 四、数据治理需求

| 要求 | 标准 |
|------|------|
| 数据分类分级 | 公开/内部/敏感/机密 四级 |
| 数据脱敏 | 手机号→138****1234 |
| 数据保留 | 对话日志 30 天，访问日志 7 天汇总 |
| 数据备份 | 每日全量 + 每小时增量 |
| 数据删除 | 用户注销后 30 天内彻底删除 |
