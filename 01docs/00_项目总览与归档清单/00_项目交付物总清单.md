## 一、项目概述

**项目名称**：索拉（Solra）——下一代去中心化自主虚拟世界

**项目定位**：大型沉浸式 3D 社交虚拟空间平台，集成 AI 虚拟人、智能推荐、实时社交、内容安全审核、商业化等核心能力。

**技术栈**：
- 协议层：Protocol Buffers / Avro / OpenAPI 3.0
- 核心引擎：C++17 / CMake / Vulkan 1.3 / Metal 3
- 业务服务：Java 17 / Spring Boot 3.3 / gRPC
- AI 服务：Python 3.11 / FastAPI / vLLM
- 客户端：Swift(iOS) / Kotlin(Android) / Vue3(Web) / Tauri2+Rust+Vue3(Desktop)
- 基础设施：Kubernetes / Terraform / ArgoCD / APISIX

**子项目（DDD 限界上下文，10 个）**：
| 缩写 | 中文名 | 领域类型 | 优先级 |
|------|--------|---------|--------|
| AVT | 虚拟人交互 | 核心域 | P0 |
| SPC | 空间消费 | 核心域 | P0 |
| CRT | 空间创作 | 支撑域 | P1 |
| SOC | 社交在场 | 支撑域 | P1 |
| GRW | 用户成长 | 支撑域 | P1 |
| MON | 商业化 | 支撑域 | P2 |
| AUTH | 身份认证 | 通用域 | P0 |
| SAF | 内容安全 | 通用域 | P0 |
| NOT | 通知消息 | 通用域 | P1 |
| INF | 基础设施 | 通用域 | P0 |

**开发进度**：10/10 子项目、78/78 功能 DDD 四层代码就绪、单元测试就绪（截至 2026-06-30）。
