## 六、成本控制策略

| 策略 | 措施 |
|------|------|
| 端侧推理优先 | 端侧 SLM 分担 50%+ 请求，减少云端 GPU 用量 |
| FinOps 监控 | 实时成本核算，按功能/用户/空间多维归因 |
| 弹性伸缩 | K8s HPA + Karpenter GPU 节点自动扩缩 |
| 多云比价 | 阿里云 + 腾讯云双供应商，避免锁定 |
| 开源优先 | 使用 Apache 2.0 协议的开源模型和框架 |
