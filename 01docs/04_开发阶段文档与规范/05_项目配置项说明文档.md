## 1. 配置管理概述

### 1.1 配置层次

```
应用配置 (application.yml) → 本地开发
     ↓
Nacos配置中心 → 环境配置 (dev/staging/prod)
     ↓
K8s ConfigMap/Secret → 容器运行时配置
     ↓
HashiCorp Vault → 敏感配置（密钥/证书）
```

### 1.2 配置优先级

```
命令行参数 > 环境变量 > Nacos配置 > application-{profile}.yml > application.yml
```

## 2. 核心配置项

### 2.1 数据库配置

```yaml
spring:
  datasource:
    url: jdbc:postgresql://${DB_HOST:localhost}:${DB_PORT:5432}/${DB_NAME}
    username: ${DB_USER:solra_app}
    password: ${DB_PASSWORD}
    hikari:
      minimum-idle: 5
      maximum-pool-size: 20
      connection-timeout: 30000
      idle-timeout: 600000
      leak-detection-threshold: 60000
  flyway:
    enabled: true
    baseline-on-migrate: true
```

### 2.2 Redis配置

```yaml
spring:
  data:
    redis:
      cluster:
        nodes: ${REDIS_NODES:localhost:6379}
      password: ${REDIS_PASSWORD}
      timeout: 3000ms
      lettuce:
        pool:
          max-active: 16
          max-idle: 8
          min-idle: 4
```

### 2.3 Kafka配置

```yaml
spring:
  kafka:
    bootstrap-servers: ${KAFKA_BOOTSTRAP:localhost:9092}
    producer:
      acks: all
      retries: 3
      compression-type: snappy
    consumer:
      group-id: ${spring.application.name}
      enable-auto-commit: false
      max-poll-records: 50
```

### 2.4 安全配置

```yaml
solra:
  security:
    jwt:
      secret: ${JWT_SECRET}                     # Vault管理
      access-token-expiration: 900              # 15分钟(秒)
      refresh-token-expiration: 604800          # 7天(秒)
      issuer: solra-auth
    cors:
      allowed-origins:
        - https://solra.io
        - https://dev.solra.io
        - http://localhost:5173
      allowed-methods: GET,POST,PUT,PATCH,DELETE
      allowed-headers: Authorization,Content-Type
```

```yaml

      endpoint: ${MINIO_ENDPOINT:http://localhost:9000}
```

### 2.7 WebSocket配置

```yaml

      heartbeat-interval: 30s
```

```yaml

  endpoints:
    web:
      exposure:
        include: health,metrics,prometheus

        enabled: true
```

## 4. 配置热更新

通过Nacos配置中心支持的动态刷新：

```java
@RestController
@RefreshScope  // 支持配置热更新
public class ConfigController {

    @Value("${solra.ai.llm.temperature}")
    private double temperature;

    @GetMapping("/config/temperature")
    public double getTemperature() {
        return temperature;
    }
}
```

热更新步骤：
1. 登录Nacos控制台
2. 修改配置项
3. 发布配置
4. 应用自动刷新（无需重启）

## 5. K8s Secret管理

```yaml
apiVersion: v1
kind: Secret
metadata:
  name: solra-secrets
type: Opaque
stringData:
  db-password: "${DB_PASSWORD}"
  jwt-secret: "${JWT_SECRET}"
  minio-access-key: "${MINIO_ACCESS_KEY}"
  minio-secret-key: "${MINIO_SECRET_KEY}"
```

> **安全要求**：Secret不提交到Git仓库，由Vault或CI注入。
