## 2. Java公共组件 (solra-common)

### 2.1 solra-common-core

**UUID v7生成器**：
```java
// 时间有序UUID，索引友好
UUID id = UuidV7Generator.generate();
```

**通用响应封装**：
```java
// 统一API响应
ApiResponse.success(data);
ApiResponse.error(ErrorCode.NOT_FOUND, "资源不存在");
ApiResponse.page(items, pagination);
```

**分页工具**：
```java
PageResult<T> result = PageUtils.toPageResult(
    springPage,
    items -> converter.convert(items)
);
```

**JSON工具**：
```java
// Jackson配置好的ObjectMapper
String json = JsonUtils.toJson(object);
T obj = JsonUtils.fromJson(json, Type.class);
```

**全局异常处理器**：
```java
@RestControllerAdvice

    // 自动将BusinessException转换为标准错误响应

```

**请求日志拦截器**：
```java
// 自动记录: 请求路径/耗时/状态码/用户ID/TraceID
@Component
public class RequestLoggingInterceptor implements HandlerInterceptor { ... }
```

**当前用户注入**：
```java
@GetMapping("/me")
public UserInfo getCurrentUser(@CurrentUser UserContext user) {
    // 自动从JWT中解析注入
    return userService.getById(user.getUserId());
}
```

**请求ID过滤器**：
```java
// 提取/生成 X-Request-ID，注入MDC
```

### 2.3 solra-common-security

**JWT工具**：
```java
// 签发Token
TokenPair tokens = JwtProvider.createTokens(user);

// 验证Token
Claims claims = JwtProvider.validateToken(token);

// 提取用户上下文
UserContext user = SecurityUtils.getCurrentUser();
```

**权限注解**：
```java
@PreAuthorize("hasRole('CREATOR')")
@PostMapping("/spaces")
public Space createSpace(...) { ... }

@PreAuthorize("hasPermission('content:review')")
@GetMapping("/admin/review-queue")
public Page<ReviewItem> getQueue(...) { ... }
```

### 2.4 solra-common-messaging

**事件发布工具**：
```java
// 发布领域事件
eventPublisher.publish(new UserRegisteredEvent(user));

// 自动设置 eventId/aggregateId/timestamp
```

**幂等消费抽象**：
```java
@KafkaListener(topics = "solra.user.events")
public void handle(ConsumerRecord<String, String> record) {
    idempotentConsumer.consume(record, event -> {
        // 自动去重，保证幂等
        processEvent(event);
    });
}
```

## 4. Python公共库 (solra-py-common)

### 4.1 核心模块

```python
# solra_py_common/

# 配置管理
from solra_py_common.config import settings  # pydantic-settings

# 日志
from solra_py_common.logging import get_logger
logger = get_logger(__name__)

# 数据库
from solra_py_common.database import AsyncSession, get_db

# Redis
from solra_py_common.cache import redis_client

# HTTP客户端
from solra_py_common.http import HttpClient
async with HttpClient() as client:
    response = await client.get("https://api.example.com")

# 异常
from solra_py_common.exceptions import NotFoundException, BusinessException

# 模型推理
from solra_py_common.inference import ModelManager
model = ModelManager.load("llm-13b")
result = await model.infer(prompt)
```

## 6. 使用规范

| 规范 | 说明 |
|------|------|
| 禁止重复造轮子 | 先查找公共组件是否有现成实现 |
| 新增公共组件 | 需经过架构组评审 |
| 版本管理 | 公共库独立版本号，变更需通知 |
| 文档 | 每个公共组件需有README + 使用示例 |
| 废弃 | 废弃需标注@Deprecated + 提供迁移指南 |
