# 08_数据备份与恢复方案

> **文档编号**：06-08 | **版本号**：v1.0 | **编制人**：运维组 | **状态**：已发布 | **更新日期**：2026-06-30

## 1. 备份策略

| 数据 | 类型 | 频率 | 保留 | RPO |
|------|------|------|------|-----|
| PostgreSQL | 全量 | 每日02:00 | 30天 | <24h |
| PostgreSQL | WAL | 持续 | 7天 | <1min |
| Redis | RDB | 每6h | 7天 | <6h |
| MinIO | 跨Region复制 | 实时 | 永久 | 0 |
| Milvus | 元数据 | 每日 | 14天 | <24h |

## 2. 备份验证

- 每日自动校验备份完整性
- 每月恢复演练

## 3. 恢复流程

```bash
# 全量恢复
pg_restore -h $DB_HOST -U solra_app -d auth_db -c latest_backup.dump

# 时间点恢复(PITR)
pg_restore + WAL回放到指定时间点
```

## 4. RTO/RPO

| 场景 | RTO | RPO |
|------|-----|-----|
| 单表误删 | <1h | <1min (WAL) |
| 全库灾难 | <4h | <1h |
