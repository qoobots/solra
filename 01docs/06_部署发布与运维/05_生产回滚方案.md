## 1. 回滚触发条件

| 条件 | 级别 | 自动/手动 |
|------|------|-----------|
| 错误率 > 5% (持续5分钟) | P1 | 自动 |
| P99延迟 > 3倍基线 | P1 | 自动 |
| 核心功能不可用 | P0 | 手动(立即) |
| 数据不一致 | P0 | 手动(立即) |

## 2. 回滚步骤

### K8s回滚
```bash
# 查看历史版本
kubectl rollout history deployment/auth-service -n solra

# 回滚
kubectl rollout undo deployment/auth-service -n solra

# 验证
kubectl rollout status deployment/auth-service -n solra
```

### 数据库回滚
```bash
# Flyway Undo (生产慎用)
./gradlew flywayUndo -Dflyway.target=2

# 或从备份恢复
pg_restore -h $DB_HOST -U solra_app -d auth_db backup.dump
```
