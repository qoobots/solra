## 1. 监控架构

```
应用层 → Micrometer → Prometheus → Grafana (可视化)
日志层 → Fluent Bit → Loki → Grafana (检索)
追踪层 → OpenTelemetry → Jaeger (调用链)
告警层 → AlertManager → 电话/短信/IM
```

## 2. 四大黄金信号

| 信号 | 指标 | 告警阈值 |
|------|------|----------|
| 延迟 | P99响应时间 | > 3倍基线 |
| 流量 | QPS | 偏离预测30% |
| 错误 | 错误率 | > 5% |
| 饱和度 | CPU/内存/连接 | > 85% |

## 4. 告警规则

| 级别 | 条件 | 通知 |
|------|------|------|
| P0 | 核心服务Down | 电话+IM |
| P1 | 错误率>5% | 短信+IM |
| P2 | CPU>85% | IM |
| P3 | 非关键异常 | 邮件 |
