# 02_生产环境部署手册

> **文档编号**：06-02 | **版本号**：v1.0 | **编制人**：运维组 | **状态**：已发布 | **更新日期**：2026-06-30

---

## 1. 生产环境拓扑

3个Region（CN主/SG/US），每Region 2个AZ，K8s集群12+节点。

## 2. 部署前检查

- [ ] TLS证书已配置
- [ ] 数据库密码已更新
- [ ] JWT密钥已配置
- [ ] 第三方API密钥已配置
- [ ] DNS已指向Load Balancer
- [ ] 监控告警已配置

## 3. 发布流程

```
1. 构建镜像 → Harbor Registry
2. 更新K8s Deployment镜像版本
3. 执行金丝雀发布（5%→20%→50%→100%）
4. 每阶段观察5-15分钟
5. 异常自动回滚
```

## 4. 健康检查

```bash
# 检查所有Pod状态
kubectl get pods -n solra

# 检查服务可用性
curl https://api.solra.io/health

# 检查数据库连接
kubectl exec -it deployment/auth-service -- curl localhost:8080/actuator/health
```

## 5. 回滚

```bash
# 回滚到上一个版本
kubectl rollout undo deployment/auth-service -n solra

# 回滚到指定版本
kubectl rollout undo deployment/auth-service --to-revision=3 -n solra
```
