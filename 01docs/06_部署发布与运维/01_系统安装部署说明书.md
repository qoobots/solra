> **文档编号**：06-01 | **版本号**：v1.0 | **编制人**：运维组 | **状态**：已发布 | **更新日期**：2026-06-30

## 1. 部署方式

支持两种部署方式：

### Docker Compose (单机/测试)
```bash
git clone https://github.com/solra-project/solra.git
cd solra
docker compose -f docker-compose.prod.yml up -d
```

### Kubernetes (生产)
```bash
kubectl apply -k contracts/deploy/kubernetes/overlays/production/
```

## 2. 系统要求

| 环境 | CPU | 内存 | 磁盘 | 节点数 |
|------|-----|------|------|--------|
| 最小 | 16核 | 32GB | 100GB SSD | 3 |
| 推荐 | 32核 | 64GB | 500GB NVMe | 6+ |

## 3. 前置依赖

- Kubernetes 1.29+
- Helm 3.14+
- cert-manager (TLS自动管理)
- Istio 1.22+ (可选)
- NVIDIA GPU Operator (AI节点)

## 4. 安装步骤

1. 创建命名空间：`kubectl create namespace solra`
2. 配置Secrets：`kubectl apply -f secrets.yaml`
3. 部署基础设施：`helm install infra ./charts/infra`
4. 部署微服务：`kubectl apply -k overlays/production/`
5. 验证：`kubectl get pods -n solra`

## 5. 验证清单

- [ ] 所有Pod Running
- [ ] Gateway可访问
- [ ] 数据库连接正常
- [ ] Redis集群正常
- [ ] Kafka集群正常
