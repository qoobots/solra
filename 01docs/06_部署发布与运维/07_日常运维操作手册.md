## 1. 日常检查

| 频率 | 检查项 | 命令 |
|------|--------|------|
| 每日 | Pod状态 | `kubectl get pods -n solra` |
| 每日 | 错误日志 | Grafana Loki查询 |
| 每日 | 磁盘使用率 | `df -h` |
| 每周 | 备份状态 | 检查MinIO备份文件 |
| 每月 | 证书到期 | `kubectl get certificates` |

## 2. 常用操作

```bash
# 重启服务
kubectl rollout restart deployment/auth-service -n solra

# 查看日志
kubectl logs -f deployment/auth-service -n solra --tail=100

# 扩容
kubectl scale deployment/avt-service --replicas=5 -n solra

# 进入容器
kubectl exec -it deployment/auth-service -n solra -- /bin/bash
```

## 3. 用户管理

```bash
# 封禁用户 (需要管理工具)
./tools/admin-cli user ban --userId=xxx --reason="违规内容"

# 解封
./tools/admin-cli user unban --userId=xxx
```
